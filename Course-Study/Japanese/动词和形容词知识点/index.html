    本文介绍了日语中动词和形容词的分类及其变形规则。动词分为三类：一类动词（五段动词），以う段结尾；二类动词（一段动词），以い段或え段结尾；三类动词（か变和サ变），如“来る”和“する”。动词变形包括ます形和て形，其中五段动词通过改变う段到い段加ます，一段动词直接加ます，而三类动词则有特定的变形规则。形容词分为两类，一类以い结尾，如“寒い”，二类无い结尾，如“綺麗”。一类形容词的肯定、否定和过去式变形规则也被提及。

    # 动词三分类

    所有的动词原型都是う段结尾。

    # 一类动词（五 (う) 段动词）

    1. 结尾是う段非る的动词，肯定是五段动词。
    2. あ段 + る、う段 + る、お段 + る 是五段动词
    3. 特例：
      • かく　さがす　かつ　あそぶ　よむ
      • こまる　おこる　やる
      • きる　かえる　はしる

    # 二类动词（上一段动词 (い段)、下一段动词 (え段)）

    い段 + る、え段 + る
    eg:
    おきる (い段 + る)
    たべる (え段 + る)
    みえる (え段 + る)
    きる (い段 + る)

    # 三类动词（か变、サ变）

    来 (く) る - か变
    する　- サ变

    但サ变有很多，通过 ” 名词 + する “ 作动词。
    eg: 勉強する　運動する　散歩する (一般可活用动词的名词才可加する)

    # 动词变形 原型 -> ます形

    # 五段动词（一类）

    う段 -> い段 + ます （う段变同行い段）
    eg:
    書 (か) く　-> 書 (か) きます
    吸 (す) う -> 吸 (す) います
    切 (き) る -> 切 (き) ります
    呼 (よ) ぶ -> 呼 (よ) びます

    # 上一段动词、下一段动词（二类）

    る -> ます
    eg:
    教 (おし) える -> 教 (おし) えます
    開 (あ) ける -> 開 (あ) けます
    調 (しら) べる -> 調 (しら) べます
    得 (え) る　-> 得 (え) ます

    # か变、サ变动词（三类）

    来 (く) る -> 来 (き) ます
    する　-> します
    eg:
    勉強 (べんきょう) する -> 勉強します

    # て形动词 原型 -> て形

    1. 三类动词的变形：する　-> して、来 (く) る -> 来 (き) て

    2. 二类动词的变形：る -> て
      eg: 食 (た) べる　-> 食 (た) べて 調 (しら) べる -> 調 (しら) べて

    3. 一类动词（う段动词的变形）：

    结尾 变形规则 例子 变形后
    す -> して 話 (はな) す 話 (はな) して
    く -> いて 磨 (みが) く 磨 (みが) いて
    ぐ -> いで 急 (いそ) ぐ 急 (いそ) いで
    う、つ、る -> って 洗 (あら) う 洗 (あら) って
    う、つ、る -> って 待 (ま) つ 待 (ま) って
    う、つ、る -> って やる やって
    ぶ、む、ね -> んで 遊 (あそ) ぶ 遊 (あそ) んで
    ぶ、む、ね -> んで 読 (よ) む 読 (よ) んで
    ぶ、む、ね -> んで 死 (し) ね 死 (し) んで

    特例：行 (い) く -> 行 (い) って

    # た形动词

    把て形动词的て换成た即可。

    # 形容词

    1. 一类形容词
      • 以い结尾
      • 漢字 (かんじ)　+ い

    eg: 寒 (さむ) い　暖 (あたた) かい

    1. 二类形容词
      eg: 綺麗 (きれい)　詳細 (しょうさい)　愉快 (ゆかい)

    形 1 表肯定时： い　+ です　（熱 (あつ) い　-> 熱いです）
    表否定时：い -> くない　（熱 (あつ) い　-> 熱くない　です　=　熱くありません）
    表过去时：い -> かった （熱 (あつ) い -> 熱かった）
    表过去否定时：い -> くなかった （熱 (あつ) い -> 熱くなかった）
