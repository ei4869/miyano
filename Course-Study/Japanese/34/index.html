    本文概述了日语初级下课程第34课《墙上挂着挂历》的内容，重点在于单词学习和语法点。课程内容涵盖了烧烤、赚钱、留下等动词以及日历、海报等名词的中日文对照和用法。语法方面，介绍了“他动词て形+あります”表示动作结果的存续状态，以及“动词ておきます”和“动词てみます”分别表示事先准备和尝试做某事的语法结构。此外，还讲解了“ために”的用法，表示目的，要求前后小句主语相同，如为了留学日本而攒钱，为了健康每天走路。本课程旨在帮助学生掌握这些日语单词和语法点，提高日语水平。

    # [他动] てあります

    “他动词て形 + あります “ 表示 有意 进行的动作结果的存续状态。

    （看上去是现象，实际上暗示有人做的）

    壁にカレンダー　が　掛かっています　自动词 + ている 描述现象
    壁にカレンダー　が　掛けてあります　他动词 + てある 暗示有人做

    # [动词] ておきます

    表示为作某种准备而有意识进行的动作。 —— 事先做某事，预先做某事。

    お客さんが来るまえに、部屋を　掃除し　ておきます。 （动作还没发生）-> 发生用ておきました

    # [动词] てみます

    表示尝试做某事。

    太田さんは中国語で手紙を　書い　てみました。太田先生试着用汉语写了一封信。

    みる て形：みて

    # [小句 1 (基本形)] ために、[小句 2] ① [目的];[名词]+ の + ために、[小句] [目的]

    "~ ために"　表示目的，其前后两个小句的主语相同。

    日本に　留学する　ために、お金をためています。
    健康　の　ために、毎日歩いています。
