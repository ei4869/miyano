    # 第五课：森 (もり) さんは　七時 (しちじ) に　起 (お) きます

    # 语法

    # 今 (いま)　x 時 (じ)　x 分 (ふん) です

    用于表示现在的时间。询问时间用　何時 (なんじ)
    上午八点半 -> 午前 (ごぜん)　八時半 (はちじはん)　です 不加の

    # [动] ます　/[动] ません /[动] ました /[动] ませんでした

    都是礼貌的形式。 肯定地叙述现在的习惯性动作、状态、未来的动作时用 ます　，否定用 ません；肯定地叙述过去的动作时用 ました，否定用 ませんでした。

    ます ません ました ませんでした
    働 (はたら) きます 働 (はたら) きません 働 (はたら) きました 働きませんでした (はたらきせんでした)
    休 (やす) みます 休 (やす) みません 休 (やす) みました 休 (やす) みませんでした

    疑问句则在句尾加 か

    # [名 (时间)] に　[动]

    表示动作发生的时间时，要在具体的时间词语后加助词 に
    注:

    • 叙述包含数字的时间后一定加 に，如：" 三月十四日 (さんがつじゅうよっか) に "
    • 今 (いま)　昨日 (きのう) 今日 (きょう)　明日 (あした)　毎日 (まいにち) 去年 (きょねん)　来年 (らいねん) 等词后不能加に
    • 星期后可加可不加。

    该句式可换为：[时间]、[名] は　[动]

    # [名 (时间)] から　[名 (时间)] まで　[动]

    表示某动作发生在某个期间时。
    eg: わたしは　九時 (くじ) から　働 (はたら) きます。 我从九点开始工作。

    # いつ　[动] ますか

    询问某动作或事态进行的时间。
    询问时间很具体时加に，如：何時 (になんじに)　何曜日 (なんようび) に　何日 (なんにち) に
    試験 (しけん) は　いつ　始 (はじ) まりますか。 什么时候开始考试？

    询问持续性动作或事态的起点或终点时，用 いつから　何曜日 (なんようび) まで等。

    # は

    は 还可用于对比
    eg: 小野 (おの) さんは　今日は　休みます。 小野女士今天休息。

    # 表达

    1. 毎朝 (まいあさ)　何時 (になんじ) に 起 (お) きますか
      当两个名词都表示时间，而前面的名词中包含 " 毎 (まい)"时，名词见不加の；前面的名词中不包含" 毎 (まい)" 时，可加可不加。

    2. ごろ
      接在时间之后，表示大致的时间范围。后面一般不加に
      eg: 昨日 (きのう)　12 時半 (じゅうにじはん) ごろ　寝 (ね) ました。 昨晚，十二点半睡觉。

    # 第六课：吉田…… よしだ} さんは　{来月 (らいげつ)　中国 (ちゅうごく) へ　行 (い) きます

    # 语法

    # [名 (场所)] へ　[动]

    使用 行きます　帰 (かえ) ります　等表示移动的动词时，行为的目的地用助词 へ 表示。

    # [名 (场所)] から　[动]

    使用移动动词时，移动的起点用助词 から。

    # [名 (人)] と　[动]

    和某人一起做某事。
    eg: 小野 (おの) さんは　友達 (ともだち)　と　帰 (かえ) りました。 小野女士和朋友一起回去了。

    # [名 (交通工具)] で　[动]

    交通手段用助词 で，而不使用交通工具而步行时，用歩 (ある) いて
    eg: 私は　バス　で　家へ　帰ります。 我乘公共汽车回家。

    # [名 (场所)] から　[名 (场所)] まで　[动]

    表示移动的范围时，起点から终点まで。

    # "に / で / へ / から / まで / と"+ は

    は　加在这些副词后面表示一种复合形式。

    # 表达

    1. たしか
      "好像是..."，表示不完全有把握的记忆。
      - あの　人は　だれですか。　—— たしか　李さんの　会社の　人です。

    2. 家 (いえ)　和 うち
      意指建筑物时一般用家 (いえ)，着重指家人时一般用うち。うち有 “我家” 的意思，但用家 (いえ) 时必须用 "私の家"。

    3. まっすく　帰りました
      "径直回家"，まっすく　表示 “笔直” 的意思。

    4. お疲 (つか) れ様 (さま) でした
      寒暄用语，“辛苦了”。

    # 第七课：李さんは　毎日　コーヒーを　飲 (の) みます

    # 语法

    # [名] を　[动]

    动词的对象用助词 を。

    # [名 (场所)] で　[动]

    动作的场所用助词 で　表示。 注：日语中存在的场所用に，动作的场所用で。

    # [名] か　[名]

    “或者”，对若干名词进行选择时。
    eg: 私は　毎朝 (まいあさ)　パンか　お粥 (かゆ) を　食べます。 我每天早上吃面包或粥。

    # [名] を　ください

    “请给我～”。

    # 表达

    1. "何" 的读法：なん　和 なに
    形式 例句
    なにと 何 (なに) と何 (なに) を　買いますか。
    なにが あそこに　何 (なに) が　ありますか。
    なにを 何 (なに) を　食べますか。
    なにで / なんで 何 (なに / なん) で　会社へ　いきますか。
    なんの それは　何 (なん) の　本ですか。
    なん時刻 / 星期等 李さんは　何 (なん) 時に　来ますか。

    对于で，虽然都可用，但最好用何 (なに) で。

    1. そうですか
      “是吗”。句尾降调表示理解了所听到的信息，句尾升调表示对对方内容有疑问。

    2. そうですね
      “好啊”。

    3. じゃあ
      “那么”，接过对方的话。正式场合用では。

    4. 寒暄语
      (1) 失礼します　
      “打搅了”“告辞了”。

    (2) いってらっしゃい　いってまいります / いってきます
    以还要返回该处为前提，要离开的人一般是 "いってまいります"（我走了），"いってきます" 是较为随便的说法，对方回答 “いってらっしゃい”，意为 “去吧”，其中含有盼望对方 “早点回来” 的心情。

    (3) ただいま　お帰りなさい
    回到家或单位时，回来的人说 "ただいま"(我回来了)，在家或单位的人说 "お帰りなさい"(你回来啦)

    (4) いらっしゃいませ　かしこまりました
    いらっしゃいませ 为店员对顾客说的寒暄语 “欢迎光临”。
    かしこまりました 为店员对顾客使用，“明白了”。
    いらっしゃい　语气较随便，用于欢迎朋友或亲近的人来家里做客。

    (5) すみません
    除了表示 “对不起” 等道歉的意思外，还有向别人搭话时使用的 “请问” 的意思。

    # 第八课：李さんは　日本語で　手紙 (てがみ) を　書 (か) きます

    # 语法

    # [名 (工具)] で　[动]

    で　还可用于表示其他其他手段以及原材料。
    eg: 何 (なに) で　うどんを　作 (つく) りますか。 用什么做面条？

    # [名 1 (人)] は [名 2 (人)] に [名 3 (物)] を　あげます

    あげます 相当于 “给”。1 给 2 某物。
    用于：我 (也可以是我亲近的人) 给某人某物 以及 某人给某人某物。
    eg: 私は　小野 (おの) さんに　お土産 (みやげ) を　あげます。 我送给小野女士礼物。
    小野さんは　森 (もり) さんに　チョコレートを　あげました。 小叶女士给了森先生巧克力。

    母 (はは) は　長島 (ながしま) さんに　ワインを　あげました。 母亲送葡萄酒给长岛先生。 (母亲是自己人)

    注：别人给我 (或我亲近的人) 要用くれます。eg: 王さんは　私に　本を　くれます。

    # [名 1 (人)] は [名 2 (人)] に [名 3 (物)] を　もらいます

    もらいます　相当于 “得到”。1 从 2 得到了某物。
    用于：我 (或我亲近的人) 得到了某人的某物 以及 某人得到了某人的某物。 赠送者后可接に也可接から。
    eg: 私は　小野さんに　辞書 (じしょ) を　もらいます。 我从小野女士那得到了一本词典。
    森さんは　長島さんから　パンフレットを　もらいます。　森先生从长岛先生那得到了小册子。

    弟 (おとうと) は　長島さんから　本を　もらいました。 弟弟从长岛先生那得到了一本书。

    # [名 (人)] に　会 (あ) います

    “见了某人”。见到的对象用助词に。

    # よ

    句尾升调，提醒对方注意其不知道的事。
    eg: もう　帰りましたよ。

    # もう

    “已经”。

    # 表达

    1. x から　もらいます
      从 x 接受 / 得到某物时，如果是从人得到，则可以に / から，如果是从会社 (かいしゃ)/ 学校 (がっこう) 之类的组织或团体得到时，只能用から。

    2. あげます
      送别人东西时，为了不给人强加于人的印象，可以用 "どうぞ"" どうですか " 比较适合。

    3. さっき / たった今 (いま)
      都表示离现在很近的过去，后续动词接过去式，如果说话人觉得离现在非常近，用 たった今 (刚刚)，稍前一点用 さっき (刚才)，这两种都是比较随便的说法，用于日常会话。

    4. 通讯手段
      電話 (でんわ)/ ファックス / メールを　もらいます。　收到电话 / 传真 / 邮件。
      電話を　かけます / 電話します 打电话
      ファックス / メール / 手紙 (てがみ) を　送 (おく) ります 发传真 / 邮件 / 信
      手紙を　出 (だ) します 寄信

    5. お願 (ねが) いします
      请求对方做某事时用 “名词 +(を)　お願いします”。

    6. 分 (わ) かりました
      “明白了”，还可用于对对方说的话的承诺和应答。

    7. 前にまえに　
      表示时间，“以前”，其中に有时可不用。
      eg: 前に　田中 (たなか) さんに　メールを　もらいました。 以前收到过田中先生的电子邮件。
