    本文是日语初级上单元一的教学内容，包括四课。第一课介绍基本的自我介绍和否定句型；第二课讲解物品的指示代词和所属关系；第三课涉及场所指示代词和存在句型；第四课则讲解方位词和存在句型。此外，还涵盖了礼貌表达、询问年龄、感谢等日常用语。通过这些课程，学生可以掌握基础的日语语法和日常交流表达。

    # 第一课：李 (り) さんは　中国人 (ちゅうごくじん)　です

    # 语法

    # [名] は　[名] です

    eg: 私　は　王 (おう) です。

    # [名] は　[名] ではありません

    “では”　口语中有时为 “じゃ”

    # [名] は　[名] ですか

    x 是 x 吗？ 回答为 はい　或 いいえ
    - はい、そうです
    - いいえ、ちがいます

    # [名] の [名]

    eg: 李さん　は　JC 企画 (ジェーシー企画)　の　社員 (しゃいん)　です。

    # 第二课：これは　本 (ほん) です

    # 语法

    # これ / それ / あれ　は　[名] です

    (1) A (说话人) 和 B (听话人) 相隔一段距离，面对面时：
    これ：离 A 近
    それ：离 B 近
    あれ：离两人都远
    (2) A (说话人) 和 B (听话人) 在同一位置，面向同一方向时：
    これ：离 A、B 近
    それ：离 A、B 近
    あれ：离两人都远

    # だれですか / 何 (なん) ですか

    不知什么人用 だれ；不知什么事物用 何 (なん)
    だれ 更有礼貌应该用 どなた

    # [名] の　[名] [所属]

    eg: わたし　の　かぎ。

    # この / その / あの　[名] は [名]　です

    eg: あの　ノートは　だれのですか。

    # どれ / どの [名]

    eg: 小野 (おの) さんの　机 (つくえ)　は　どの　机 (つくえ)　ですか。

    # 表达

    1. 方 (かた)
      この / その / あの　人 (ひと)　-> この / その / あの　方 (かた) 更加礼貌
    2. 询问年龄
    • 何歳 (なんさい) ですか
    • おいくつですか 比较礼貌
    • 直接问小孩子时，いくつ？　何歳 (なんさい)
    1. どうぞ
      给对方物品，劝对方进餐。
    2. どうもありがとうございます
      表示感谢，感情很强。

    # 第三课：ここは　デパート　です

    # 语法

    # ここ / そこ / あそこ　は　[名] です

    用于指示场所。

    # [名] は　[名 (场所)]　です

    表示 “名词” 存在于 “名词（场所）”。
    eg: トイレ　は　ここ　です。 厕所在这儿。

    # [名] は　どこですか

    用于询问存在的场所。
    eg: トイレは　どこですか？ ―あちらです。

    # [名] も [名]　です

    x 也是 x。

    # [名] は [名] ですか、 [名] ですか

    eg: かばん　売 (う) り場 (ば) は　一階 (いっかい)　ですか、二階 (にかい)　ですか。 卖包的柜台在一层还是二层。

    # [名] は　いくらですか

    询问价钱时，用 いくら

    # 表达

    1. こちら / そちら / あちら / どちら
      是ここ / そこ / あそこ / どこ的礼貌说法

    2. あのう
      用于向对方搭话或想引起对方注意。或者想对对方说难以启齿的事等等。

    # 第四课： 部屋 (へや)　に　机 (つくえ) と　いすが　あります

    # 语法

    # あります和います

    あります 用于不具有意志的事物
    います 用于具有意志的人、动物等

    1. [名 (场所)] に　[名 (物 / 人)] が　あります / います
      表示事物存在： x 有 x 某地有某物
      eg: ここに　本ほん　が　あります。
      公園 (こうえん) に　猫 (ねこ)　が　います

    2. [名 (物 / 人)] は [名 (场所)] に　あります / います
      表示位置： x 在 x 某物在某地
      eg: いすは　部屋 (へや) に　あります。
      子供 (こども) は　公園 (こうえん) に　います。

    之前的 x は　どこですか　可以用 x は　どこに　ありますか / いますか　表示。

    # [名] と　[名] [并列]

    表示并列，“和”

    # 上 (うえ)　下 (した)　前 (まえ)　後 (うし) ろ　隣 (となり)　中 (なか)　外 (そと)

    方位词

    # ね

    当说话人征求听话人的同意时，句尾用助词 "ね"，升调

    # 疑问词 も [动 (否定)]

    表示全面否定。
    {教室…… きょうしつ} に　だれも　いません。

    # 表达

    1. 上 (うえ)
      只表示垂直上方的范围，不能用作表示表面。

    2. ええと
      被别人问及某事时思考如何回答说的话。

    3. ご家族 (かぞく)　ご兄弟 (きょうだい)　ご両親 (りょうしん)
      说到别人亲属时的礼貌用法，指对方的家人。家族 (かぞく) 指自己的家人。

    4. 兄弟 (きょうだい)
      指同一父母的人，也可以同父或同母，包括兄弟姐妹。

    5. 一人暮 (ひとりく) らし
      指离开家一个人生活。
