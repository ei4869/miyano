    # 第二十一课：わたしは　すき焼 (や) きを　食べた　ことが　あります

    # 语法

    # 动词的た形

    即把て形的て换成た，把で换成だ。

    # [动 (た形)] ことが　あります

    表示过去的经历，即 “（曾经）~ 过”。通常用于至少半年以前发生的事情，而在叙述 “昨天” 等离现在很近的以前的经历时，不能用此句型。本句型即可用于叙述经历的时间、次数，也可用于谈论某种经历。
    否定形式是：“动词た形 + ことが ありません”。强调完全没有某种经历时，可以加 “一度 (いちど) も”。
    疑问形式是：“动词た形 + ことが ありますか” 。疑问句一般只问有无某种经历。回答时，可以重复问句中的部分词语，也可以只说 “はい ，あります”“いいえ，ありません”。

    北京へ　行った　ことが　ありますか。 你去过北京吗？
    —— いいえ，一度も　行った　ことが　ありません。 没有，一次也没去过。
    —— いいえ，ありません。 不，没有。

    # [动 (た形)] 後 (あと) で，～

    表示动作在另一个动作之后发生时，使用该句型。
    eg：映画を　見た　後で，食事 (しょくじ) を　しました。 看完电影后吃了饭。

    这个句型也可以以 “名词 + の + 後で” 的形式使用。
    eg：仕事の　後で，映画を　見ます。 工作之后看电影。

    前面学过 “前に”，后面必须是 “に”，而这里 “後で” 后面必须为 “で”，且 “～前に” 前面的动词必须为基本形，而 “～後で” 前面的动词必须为た形。

    # [动 (た形)] ほうが　いいです

    用于在两种事物中进行选择时。“最好做～”。
    否定表达形式为：“动词未然形 + ない ほうが　いいです”，也即 “动词ない形 + ほうが　いいです”，表示 “最好别那么做”。
    当建议对方做理想的动作、行为时，句尾可加 “よ”。

    eg：
    もっと　野菜を食べた　ほうが　いいですよ。 还是多吃点蔬菜好啊。
    そんなに　慌 (あわ) てない　ほうが　いいです。 别那么慌张啊。

    # [动] ましょうか [提议]

    用于提议对方和自己一起做某事或向对方提议自己为对方做某事。 另外还有向对方确认自己的提议是否可行的意思。
    何か　食べましょうか。 吃点什么吗？

    回答时，若意思为 A 向 B 提议为对方做某事时，则 B 说 “お願 (ねが) いします”。若意思为 A 向 B 提议双方一起做某事时，则 B 说 “はい，そうしましょう” 或 “いいですね”。反对或拒绝提议时则说 “～ないで　ください” 或 “いいえ，ちょっと…”。

    # 表达

    1. その　ほうが　いいです
      名词及指示事物的词也可以后续 “ほうが　いいです” 来表示比较、选择的意思。
      形式为：
    • “名词の　ほうがいいです”
    • “これ / それ / あれ　の　ほうがいいです”
    • “この / その / あの ほうがいいです”

    eg:
    速 (はや) いですから，飛行機 (ひこうき) の　ほうがいいです。 因为速度快，还是坐飞机去比较好。

    1. 何度 (なんど) か
      “几次”，在表示不定数量时，日语中疑问词后面必须加 “か”。类似还有 “いつか　どこか　何か”。
      eg：仕事の　後で，何度か　見に　行った　ことが　あります。 下班后，去看过几次。

    2. ９時過 (す) ぎ
      “～過 (す) ぎ” 用在表示时间或年龄的词语后面表示超过了该时间或年龄。用在表示时间的词语后面时，是 “数词 + 時 / 分 + 過ぎ” 的形式。用在表示年龄的词语后面时，是 “数词 + 過ぎ” 的形式，中间不加 “歳 (さい)”。
      eg：昨夜 (ゆうべ)　１０時過ぎに　地震 (じしん) が　ありました。 昨晚 10 点刚过发生了地震。

    # 第二十二课：森さんは　毎晩　テレビを　見る

    # 语法

    # 敬体形和简体形

    之前学习的 “～ます”“～です” 是对长辈或关系不太亲密的人使用的形式，为敬体形。与此相反，若对方是同辈或晚辈，或者关系较密切时则使用 “简体形”。
    动词本身具有敬体形和简体形，如敬体形包括：ます　ません　ました　ませんでした；简体形包括：动词基本形、ない形、た形。
    但一类形容词、二类形容词、名词本身没有敬体形和简体形之分，只有在做谓语时，其谓语形式才有敬体形和简体形。

    # 动词的简体形

    敬体形 简体形 敬体形 简体形
    现在将来形式 肯定 買います (买) 買う （基本形） あります (有) ある
    否定 買いません 買わない (ない形) ありません ない
    过去形式 肯定 買いました 買った （た形） ありました あった
    否定 買いませんでした 買わなかった (なかった形) ありませんでした なかった

    コーヒー，飲む？ 喝咖啡吗？
    —— ううん，飲まない。 不，不喝。 （飲みません -> 飲まない）

    注：简体的会话中常省略助词。

    # 一类形容词谓语形式的简体形

    敬体形 简体形
    现在将来形式 肯定 忙しいです (忙) 忙しい
    否定 忙しくないです 忙しくない
    过去形式 肯定 忙しかったです 忙しかった
    否定 忙しく　なかったです 忙しく　なかった

    就是去掉です即为简体形。 （形容词的肯定、否定、过去、过去否定语法见 “动词和形容词知识点”）

    昨日の　試験，どうだった？ 昨天的考试怎么样啊？
    —— ちょっと　難 (むずか) しかった。 稍微有点难。

    注：形 1 的简体形与形二以及名词不同，后面不能加 “だ”。

    # 二类形容词谓语形式的简体形

    敬体形 简体形
    现在将来形式 肯定 簡単です (简单) 簡単だ
    否定 簡単では　ありません 簡単ではない
    过去形式 肯定 簡単でした 簡単だった
    否定 簡単では　ありませんでした 簡単では　なかった

    现在将来形式的肯定疑问句用去掉简体形中的 “だ” 的形式（名词也一样）。

    この　ネクタイ，派手 (はで)？ 这台领带花哨吗？
    —— ううん，全然　派手じゃ　ないわ。 不，一点都不花哨。

    森さんの　アパート，静 (しず) か？ 森先生居住的公寓安静吗？
    —— うん，静かだよ。 嗯，很安静。

    # 名词谓语形式的简体形

    敬体形 简体形
    现在将来形式 肯定 晴 (は) れです (简单) 晴 (は) れだ
    否定 晴 (は) れでは　ありません 晴 (は) れではない
    过去形式 肯定 晴 (は) れでした 晴 (は) れだった
    否定 晴 (は) れでは　ありませんでした 晴 (は) れでは　なかった

    和第二类形容词谓语形式的简体形同种变法。

    昨日，スキー場 (じょう) は　雪だった？ 昨天滑雪场下雪了吗？
    —— ううん，雪じゃ　なかったよ。いい　天気だった。 没有下雪啊，天气挺好的。

    # [小句] けど，[小句]　[转折、铺垫]

    “けど” 连接两个句子，表示转折，起着把两个句子归纳成一个句子的作用，用于口语。

    昨日の　試験，どうだった？ 昨天的考试怎么样啊？
    —— ちょっと　難しかったけど，まあまあ　できたよ。 有点难，不过都做出来了。

    另外，“けど” 花园不表示转折而表示铺垫的用法。
    来週，送別会 (そうべつかい) を　するけど，都合 (つごう) は　どうかな？ 下星期要开一个欢送会，你时间方便吗？

    # 表达

    1. 敬体和简体
      正式场合和书信一般用敬体；与家人、朋友交谈时用简体，有关法律的文章、报纸、商业文件等也都用简体。

    お母さん，あの　玩具 (おもちゃ) が　欲しい。 妈妈，我想要那个玩具。
    昨日　駅前で　大 (おお) きい　火事 (かじ) が　あった。 昨天车站附近发生了很大的火灾。

    1. 简体疑问
      用简体表示疑问时，一般名词句、形容词句省略敬体形的 “～ですか”，句尾读升调。动词句则把动词敬体形变成简体形，去掉疑问助词 “か”，句尾读升调。另外书写时为例表示疑问需要加 “？”
      これ，日本製 (せい)？ 这个，是日本货？

    2. うん　和 ううん
      简体会话中，“はい” 说成 “うん”，“いいえ” 说成 “ううん”。

    3. 简体助词
      “かな” 和 “の” 这两个助词只能接在简体形后面。

    • 助词 “かな” 一般用于自言自语的场合，如有听话人在场则表示通过让对方听到自己的自问自答而向对方提供一种不太确定的信息。
      今日　李さんは　来 (く) るかな。 小李今天会来吗？

    • 助词 “の” 用于要求说明或确认某事。
      ああ，小野さん，どうしたの？ 小野女士，你好。有什么事吗？
      寒いの？ 冷吗？

    1. ～って [传闻]
      “太田 (おおた)，今度 (こんど)　中国へ　転勤 (てんきん) だって（听说太田要调到中国去工作）”，这里的 “転勤 (てんきん) だって” 是在简体形 “転勤 (てんきん) だ” 的后面加 “って” 表示听说、传闻。

    2. 省略 [简体句的省略]
      在口语中，简体句中的一些音往往不发，如 “森さんの　電話番号 (でんわばんご)，知ってる？”，其中省略了 “ている” 的 “い”，“電話番号を” 中的助词 “を” 也省略了。

    3. “わ” 和 “よ”
      都可用于简体句或敬体句句尾。“便利よ”“大きいですわ”。

    4. 电话号码的读法
      数字要一个一个读。

    5. “～方 (かた)”
      动词 “ます” 形去掉 “ます” 的形式加上 “方 (かた)” 就有 “~ 的做法” 的意思，用来表示动作的方式。

    ます形 去掉ます ~ 方 (かた) 做法
    話 (はな) します 話し 話し方 (かた) 说法
    読 (よ) みます 読み 読み方 (かた) 读法
    作 (つく) ります 作り 作り方 (かた) 做法
    やります やり やり方 (かた) 办法
    使 (つか) います 使い 使い方 (かた) 用法

    # 第二十三课：休みの　日 (ひ)，散歩したり　買い物に　行ったり　します

    # 语法

    # [动] たり [动] たり します

    用于列举若干种有代表性的动作时，即 “动词た形 + り　します”，“します” 需要根据上下文判断时态。

    吉田 (よしだ) さんは　日曜日 (にちようび) には　何を　しますか。 吉田先生，你星期天都做什么？
    —— 家の　掃除 (そうじ) を　したり　洗濯 (せんたく) を　したり　します。 打扫打扫房间，洗洗衣服。
    （する -> して -> した -> したり）

    注：动词た形即动词简体的过去式。

    # [形 1] かったり　[形 1] かったりです；[形 2] だったり　[形 2] だったりです；[名] だったり　[名] だったりです

    用于表示状态或状况有多种可能时，其构成形式是把形 1 た形、形 2 た形、名词た形之后加り。

    飛行機の　チケットは　高かったり　安かったりです。 飞机票有时贵，有时便宜。
    この公園は　時間に　よって　静だったり　にぎやかだったりです。 根据时间，这个公园有时寂静有时热闹。
    日本語の　先生は　中国人だったり　日本人だったりです。 日语老师既有中国人也有日本人。

    # [小句] + か

    用于表示某种不确定的内容。

    # [小句] + か　どうか

    1. 将不包含疑问词的疑问句作为一个长句中的一部分时，将动词、一类形容词的敬体形变成简体形，并在后面加上 “か　どうか”。

    eg：
    私は　今年 (ことし) の　夏，北京へ　行きますか。+ 私は　分かりません。
    可以简化为：私は　今年の　夏，北京へ　行くか　どうか　分かりません。 我今年夏天去不去北京，还不知道。

    私は　この料理が　辛いか　どうか　知りません。 我不知道这道菜辣不辣。

    小野さんは　仕事が　終 (お) わったか　どうか　聞 (き) いて　ください。 问一下小野女士，工作完了没有。

    1. 另外，也可以重复使用动词的 “基本形” 和 “ない形”，表达和上面相同的含义。
      私は　今年の　夏，北京へ　行くか　行かないか　分かりません。
      私は　この　料理が　辛いか　辛くないか　知りません。

    2. 如果 “～か　どうか” 的前面为名词或二类形容词时，“～か　どうか” 直接接在名词或二类形容词后，其间不加 “だ”。另外，“～か どうか” 小句中的主语后面必须用 “が”，而不能用 “は”。

    私は　キムさんが　中国人か　どうか　知りません。 我不知道金女士是不是中国人。
    私は　キムさんが　中国人か　中国人では　ないか　知りません。

    # [疑问词小句] + か

    将包含 “何 (なに)　誰 (だれ)　どこ” 等的疑问句，作为一个长句中的一部分使用时，需将疑问词小句的动词、一类形容词的敬体形变成简体形，再加 “か”。
    かぎが　どこに　あるか　教えて　ください。 请告诉我钥匙在哪？ あります + か -> ある + か
    昨日　何を　食べたか　忘れました。 忘记昨天吃什么了。
    どの　料理が　辛いか　知りません。 不知道哪道菜辣。

    与上面的 “～か　どうか” 相同，“か” 前面出现名词或二类形容词时，“～か　どうか” 直接接在名词或二类形容词后，其间不加 “だ”。
    この　歌 (うた) が　誰の　歌 (うた) か　知って　いますか。 你知道这是谁的歌吗？

    # 表达

    1. ～に　よって
      “根据...”，在句子根据上下文灵活翻译。也可用 “～に　よります” 的形式用于句尾。
      結婚式 (けっこんしき) の　やり方は　国 (くに) に　よって　違 (ちが) います。 结婚典礼的形式因国别而异。

    この　電車は　いつも　込 (こ) みますか。 这电车总是很挤吗？
    —— いえ，時間に　よります。 不，分时间。

    1. 高かったり　安かったり，いるいる　あります
      一类形容词、二类形容词以及名词后续 “たり” 时，句尾并不是总要后续 “です”。可根据情况接其他形式。
      eg：通勤 (つうきん) は　電車だったり　車だったり，日に　よって　違います。 有时坐电车上班，有时开车上班，不一定。

    2. あっ，そうだ
      突然想到某事，或突然想起某事时的表达，“啊，对了”，“啊，想起来了”。

    3. 浴衣 (ゆかた)
      浴衣，日本典型的夏季和服。

    # 第二十四课：李さんは　もう　すぐ　来 (く) ると　思 (おも) います

    # 语法

    # [小句 (简体形)] と　思います

    表示说话人的思考内容时，在句子后面加 “と　思います”，“～と　思います” 前的小句谓语必须是简体形。

    田中さんは　来 (こ) ない　と思います。 我想田中先生不会来。 (动词)
    日本は　きれいだ　と思います。 我觉得日本很干净。 (形 2)
    明日は　いい　天気だ　と思います。 我觉得明天会是个好天气。 （名词）

    注：一般习惯把小句谓语和と连起来读，即 “天気だと”。

    “～と　思います”（思考的根据为现在时）只能表示说话人的思考内容，但 “～と　思いました”（发表感想或曾经认为怎么样）和 “～と　思っています”（思考的根据为过去时）既可以表示说话人的思考内容，也可以表示说话人以外的人的思考内容。

    馬 (ば) さんは　日本へ　行きたい　と　思って　います。　　小马想去日本。

    # [名 (人)] は　[小句 (简体形)] と　言 (い) いました

    向第三者转述他人所说的话时，在所说的话后加 “～と　言 (い) いました”。
    陳 (ちん) さんは　パーティーに　行く　と　言 (い) いました。 老陈说要去参加联欢会。

    如果想明确指出是向谁说的，则在句子中加入 “名词 + に”。
    小野さんは　李さんに　ちょっと　休みたい　と　言 (い) いました。 小野女士跟小李说想休息一下。

    如果转述某人反复说起的事情时，用 “～と　言って　います”。
    来週の　ハイキングに　張 (ちょう) さんも　行きたい　と　言って　います。 小张说他也想参加下星期的郊游。

    注：“～と　思います” 和 “～と　言いました” 前面出现名词或二类形容词小句时必须加 “だ”。
    あそこは　駅だと　思います。 我想那儿是车站。

    # ～のです　/ んです

    1. ～のです　/ んです
      “～のです　/ んです” 表示所讲的内容与前句或前项内容有所关联。用于说明状况或解释原因、理由。“～のです” 多用于书面语，而 “～んです” 是 “～のです” 的口语形式。“～のです　/ んです” 前接简体形，但是名词和二类形容词的现在将来形的肯定形，“だ” 要换成 “な”。

    すみません，頭が　痛 (いた) いんです。 对不起，我头疼。 （形 1）

    パソコンは　止 (や) めた　ほうがいいですよ。 烟还是戒了为好。
    —— そうなんですが，やめる　ことが　できないんです。 是啊，但就是戒不掉。 （动词）

    すごい　人気にんきですね。 他可真受欢迎啊。
    —— ええ，あの　歌手 (かしゅ) は　日本では　とても　有名 (ゆうめいなん) です。 是，那个歌手在日本非常有名。 （名词）

    1. どうして　～のですか / んですか
      前面学过询问理由的 “どうしてですか”。询问理由更完整的形式是 “どうして　～のですか / んですか”。回答词问句时也要用 “～のです / んです” 的形式。

    どうして　食べないんですか。 为什么不吃？
    —— もう　お腹 (なか) が　いっぱいなんです。 已经吃饱了。

    # [小句] が，[小句] [铺垫]

    "が" 用于提示接下来要说的内容，即铺垫。与前面学过的表示铺垫的 “けど” 相比，“が” 多用于书面语。

    李さんを　探 (さが) して　いるんですが，どこに　いますか。 我在找小李，她在哪儿。

    # 表达

    1. ～と　思います
      用于向别人陈述自己的想法。

    2. どうやって～ [询问方法]
      “どう” 和动词 “やる” 的て形 “やって” 组成。一般询问方法时用 “どうやって”，而询问理由时则用 “どうして”。

    東京タワーへ　行きたいんですが，どうやって　いきますか。 我想去东京塔，请问怎么走？

    1. ～に　ついて
      “关于～”，一般后续 “考 (かんが) えます（思考）”“思います（想）”“研究 (けんきゅう)（研究）” 等表示思维活动的词语。用作标题时，如 “日本の　音楽に　やって（关于日本音乐）” 时，一般省略后面的动词。

    2. とうとう
      副词，相当于 “终于”“结局”“到底” 等，表示不论结果好坏，想到的事态经过一定的阶段终于实现了。

    とうとう お別 (わか) れですね。 终于要分别了。

    1. 分别的寒暄语
    • お世話 (せわ) に　なりました ：收到别人的照顾和帮助，分手时所说的表示感谢的话。
    • お世話になります　：今后将要得到别人的帮助或指导时使用。
    • ～に　よろしく　お伝 (つた) え　ください :“请向～问好”，如果双方关系亲密，则直接 “～に　よろしく”。
    • お元気 (げんき) で : “请多保重”，一般用于时间较长的分别，意为祝愿对方身体健康，对方必须是健康状况良好的人，“お大事 (だいじ) に（请多保重）” 则只能对生病的人或受伤的人使用。
      お気 (き) をつけて ： “小心”，用于一般性分别，意为祝愿对方路途平安。
    1. 出張中 (しゅっちょうちゅう)
      表示某个动作正在进行用 “～中 (ちゅう)”，一类、二类动词后续 “中 (ちゅう)” 时，用 “ます形” 去掉 “ます” 的形式。三类动词后续 “中 (ちゅう)” 用 “ます形” 去掉 “します” 的形式。三类动词 “来 (き) ます” 不能后续 “中 (ちゅう)”。另外并非所有的一类、二类动词都可以后续 “中 (ちゅう)”。

    森さんに　何度も　電話を　かけましたが，話 (はな) し中 (ちゅう) でした。 给森先生打了好多次电话，一直占线。
