    # 第十七课：わたしは　新 (あたら) しい　洋服 (ようふく) が　欲 (ほ) しいです

    # 语法

    # [名] が　欲しいです

    表示愿望时使用。
    eg：わたしは　新 (あたら) しい　洋服 (ようふく) が　欲 (ほ) しいです。　我想有套新西服。

    # [名] を　[动] たいです

    表示 “想～”，“名 1 は　名 2 を　～たいです”。名词 1 为愿望的主体，名词 2 表示希望进行动作的对象。“たい” 前接动词 “ます形” 去掉 “ます” 的形式。
    使用 “欲しいです” 或 “～たいです” 时，如果是陈述句，则第一人称 “わたし” 为主语。如果句子是疑问句，则第二人称 “あなた” 是主语，这两种情况的主语通常可以从句子前后关系判断出来，常省略。 注：不能用在第三人称上。
    eg: 今日は　お酒を　飲みたく　ないです。 今天不想喝酒。

    注意： 如 “わたしは　お茶が　飲みたいです” 所示，“たい” 的对象有时不用 “を”，而用 “が” 表示，但 “欲しい” 的对象只能用 “が” 表示。
    水を　飲みたいです。
    水が　飲みたいです。
    水が　欲しいです。

    1. 全面否定 “名词 + を + 动词” 时，使用：“疑问词 + も + 动词否定形式”。
    2. 全面否定 “名词 + に / から / と + 动词” 时，使用：“疑问词 + にも / からも / とも + 动词否定形式”。
    3. 全面否定 “名词 + へ + 动词” 时，既可以使用 “疑问词 + も + 动词否定形式”，也可以使用 “疑问词 + へも + 动词否定形式”。

    eg: 誰に　会いたいですか。 —— 誰にも　会いたくないです。 你想见谁？ 谁也不想见。

    # [动] ませんか

    动词的否定形式 “~ ませんか”，表示一种提议。“做某事怎么样”。

    # [动] ましょう

    把动词 “ます形” 去掉 “ます” 换成 “ましょう”，可表示提议。
    eg: ちょっと　休みましょう。　休息一下吧。

    “ましょう” 的礼貌程度没有 “ませんか” 高。

    # 疑问词 + でも

    表示在任何情况下事态都相同时使用。
    eg:
    誰も　分 (わ) かります。 　谁都明白。
    誰も　分 (わ) かりません。 谁都不明白。

    注：いつも表示 “常常，经常” 的意思，而いつでも表示 “不论任何时候”。
    いつでも　行きます。　随时去或有必要的话任何时候都去。
    いつも　行きます。 经常去。

    # ね

    缓和语气。

    # 表达

    1. “欲しいです” 和 “飲みませんか” 愿望、希望和邀请
      “何が　欲しいですか”“何を　食べたいですか” 等直接询问他人的愿望，可用于朋友，上级对下级之间，但如果对方是上级、长辈，应使用 “どうですか”“～ませんか” 等。

    另外，别人邀请自己参加某一活动时，可用 “ましょう” 回答，“ましょう” 有一起、共同做某事的含义，既可以用于提议，也可以用于对别人提议的回答。

    1. 今年 (ことし) 中 (じゅう) に
      “~ 中 (じゅう) に” 表示该期间结束之前的意思，一般接在表示未来某一特定时间的词后，如 “今日”“明日”“今週 (こんしゅう)”“来週 (らいしゅう)”“今年 (ことし)” 等后面。“今日”“明日”“今年 (ことし)” 后续 “~ 中 (じゅう) に” 时，读 “じゅうに”，“~ 週 (しゅう)”“~ 月 (がつ / げつ)” 后续 “~ 中 (ちゅう) に” 时，读 “ちゅうに”。

    eg: 今年 (ことし) 中 (じゅう) に 結婚したいです。 我想今年结婚。

    1. ぜひ
      “一定，必定”，后接 “~ たい”“~ てください”“~ ましょう” 等，表达强烈愿望、意志。

    # 第十八课：携帯電話 (けいたいでんわ) は　とても　小 (ちい) さく　なりました

    # 语法

    # [形 1] なります

    表示性质或状态的变化，构成方式是：形 1 的词尾 “い” 变成 “く” 再加 “なります”。
    “自然地变化”

    eg: パソコンは　安 (やす) く　なりました。 个人电脑便宜了。

    # [形 1] します

    表示因主语的意志性动作、作用而引起事物发生变化的场合。构成方式是：形 1 的词尾 “い” 变成 “く” 再加 “します”。
    “人为地变化”

    eg: ジュースを　冷 (つめ) たく　します。 把果汁冰镇一下。

    # [形 2]/[名] に　なります

    同语法一，同样表示性质或状态的变化，构成形式：直接在形 2 和名词后加 “になります”。
    “自然地变化”

    eg:
    息子 (むすこ) は　医者 (いしゃ) に　なりました。 儿子当医生了。
    もう　元気に　なりました。 已经恢复健康了。

    # [形 2]/[名] に　します

    同语法二，同样表示因主语的意志性的动作、作用等引起事物发生变化。
    “人为地改变”

    eg:
    部屋を　綺麗 (きれい) に　して　ください。 请把房间打扫干净。
    会議室 (かいぎしつ)　禁煙きんえんに　します。 会议室里禁烟。

    注：在餐馆里回答服务员的点餐询问时，可以使用 “~ に　します” 这一形式。表示把自己决定的事情告诉对方。“コーヒーに　します（我要咖啡。）”

    # [形 1]/[形 2] ほうが　いいです

    比较两个以上事物的性质，认为其中一个比较好。 形 2 要多加一个 “な”。
    eg:
    部屋は　広い　ほうが いいです。 房间还是宽敞一点好。
    子供は　元気な　ほうが　いいです。 孩子还是精神一点好。

    # 自动词和他动词

    1. 自动词：现象、状态 是谁做的？（和我没关系）
      不带宾语的动词，不考虑外力影响的情况下，主语自主地进行动作，或自然地发生变化。主语用 “が” 表示。

    窓 (まど) が 開 (あ) きました。 窗子开了。
    タイムサービスが　始はじまります。 时段廉价销售就要开始了。

    1. 他动词：动作、行为 强调是谁做的
      主语作用于宾语使其发生变化，宾语用 “を” 表示
      (李さんが) 窓 (まど) を 開 (あ) けました。 (小李) 打开窗户了。
      (デパートが)　タイムサービスを　始はじめます。 百货公司开始时段廉价销售。

    日语的一部分自、他动词在词形上有对应关系，即需要多记单词，作为自动词和他动词时单词形式可能不同。

    本课中 “~ なります” 为自动词，“~ します” 为他动词。

    注：特殊情况：当自动词表示移动行为时，移动的 “经过点” 和 “起点” 用助词 “を” 表示，这里的 “を” 不表示宾语。
    eg:
    公園を　散歩する。 （移动性自动词）
    学校 (がっこう) を　卒業 (そつぎょう) する。 （离开的场所）
    パレーを　踊 (おど) る。 （解释一：同族目的语，可用を，因为芭蕾是舞蹈，踊 (おど) る是跳舞，所以用を ——“踊りを　踊る”；解释二：古日语里踊る为移动性动词）

    # 表达

    1. 声 (こえ)　和 音 (おと)
      人和动物发出的声音为声 (こえ)；物体发出的声音为音 (おと)。类似英语中 voice 和 sound。

    2. まとめて
      “汇总”。
      eg: お正月 (しょうがつ) は　洋服が　安 (やす) く　なりますから，まとめて　買います。 过年时，西式服装降价，所以我一下子买很多。

    3. 三割引 (さんわりびき)
      表示 “七折” 的意思，即降价三成。
      “八五折” 即 “15(じゅうご)％(パーセント)　引 (びき)”。

    4. 似合 (にあ) います
      “似合 (にあ) います” 为自动词，表示 “相称”“适合”。
      eg: 赤い　ほうが　いいですね。よく　似合 (にあ) いますよ。 红的好吧，很适合你。

    5. “間 (ま) もなく”　和 “もう　よぐ”
      两者都是副词，表示 "一会儿"" 不久 "，但 “間 (ま) もなく” 更郑重些，用于书面语，而 “もう　よぐ” 多用于口语。
      eg:
      間 (ま) もなく　タイムサービスの　時間です。 时段廉价销售时间马上就要到了。
      もうよぐ　六時に　なります。 马上就到六点了。

    6. “さらに”　和 “もっと”
      都表示 “更加” 的意思，但 “さらに” 更郑重些，而 “もっと” 较随便。

    7. さあ
      用于向对方提出建议或促使对方做某事，只用于口语。
      eg：さあ，今から　試験 (しけん) を　始めますよ。準備 (じゅんび) は　いいですか。 好了，现在开始考试，准备好了吗？

    # 第十九课：部屋の　かぎを　忘れないで　ください

    # 语法

    # 动词的ない形

    1. 动 1：把ます形去掉ます后的最后一个音变成相应的あ段音，再加ない。如果最后音为い时，则い变わ加ない。
      （或者，对于动词原型，结尾う段变あ段加ない即可，但对于う结尾，う变わ，再加ない）
    2. 动 2：把ます形去掉再加ない。（或者，对于动词原型，る变ない）
    3. 动 3：把 “来 (き) ます” 变成 “来 (こ) ない”，把 “します” 变成 “しない”。
    类别 示例
    一类动词 書く → 書 (か) きます → かき → かかない
    急ぐ → 急 (いそ) ぎます → いそぎ → いそがない
    飛ぶ　→　飛 (と) びます → とび → とばない
    読む　→　読 (よ) みます → よみ → よまない
    死ぬ　→　死 (し) にます → しに → しなない
    待つ　→　待 (ま) ちます → まち → またない
    売る　→　売 (う) ります → うり → うらない
    買う　→　買 (か) います → かい → かわない
    話す　→　話 (はな) します → はなし → はなさない
    二类动词 食べる　→　食 (た) べます → たべ → たべない
    見る　→　見 (み) ます → み → みない
    三类动词 来る　→　来 (き) ます → き → こない
    する　→　します → し → しない

    # [动] ないで　ください

    表示否定的命令。即 “动词未然形 + ないで　ください” 或者 “动词ない形 + でください”。动词未然形是ない形去掉ない的形式。
    eg：無理 (むり) を　しないで　ください。 请不要勉强。

    # [动] なければ　なりません

    表示必须，“未然形 + なければ　なりません” 或者 “未然形 + ないと　いけません”。
    eg: すぐに　書類 (しょるい) を　送 (おく) ら　ないと　いけません。 得马上把文件寄出去。
    この　薬 (くすり) は　毎日　飲まなければ　なりません。 这个药必须每天吃。

    # [动] なくても　いいです

    表示不做某事也可以。“未然形 + なくても　いいです”。
    eg: 慌 (あわ) て　なくても　いいですよ。　 不要慌慌张张的。

    # [名 1] が　[名 2] です

    区别：
    旧信息 は　新信息 です
    新信息 が　旧信息 です

    其中旧信息指的是话题的对象。

    山田 (やまだ) 先生は　どの　方 (かた) ですか。
    —— 山田 (やまだ) 先生は あの方 (かた)　です。
    —— あの　方 (かた) が　山田 (やまだ) 先生です。
    问题提到了话题即 “山田 (やまだ) 先生”，故 “山田 (やまだ) 先生” 为旧信息。

    注：叙述直接观察到的某事物，而不是作为话题谈论该事物时，使用 “[名词] が　[动词]” 的形式。因为这里同样是新信息作主语。
    eg：子供が　公園で　遊 (あそ) んで　います。 孩子在公园里玩。
    パスが　来 (き) ました。 公共汽车来了。

    注：“何 (なに)　誰 (だれ)　どこ　いつ” 等表示疑问的词作主语时不能用 “は”，只能用 “が”，因为这些词不能表示旧信息。
    eg: 何が　好きですか。 喜欢什么？

    # 表达

    1. "~ なければ　なりません" 和 "~ ないと　いけません"
      两者都表示义务、必要性，但"~ なければ　なりません "更正式，用于书面语，"~ ないと　いけません "多用于口语，并且可以省略为 “"~ ないと”
      eg: あっ，８時ですね。もう　駅へ　行かないと。 啊，8 点了，我得去车站了。 （っ（ltu））

    2. 初心者 (しょしんしゃ)　上級者 (じょうきゅうしゃ)
      初心者 (しょしんしゃ) 表示 “初学者”。在日语中，表示技能等级为高级时，使用 “上級 (じょうきゅう)”，而 “高級 (こうきゅう)” 不用于描述人的技能、技术等，只限于表示高质量的物品。

    3. 何度 (なんど) も
      "何 + 量词 + も + 肯定形式" 表示数量多。
      eg：何度も　転 (ころ) びましたが，とても　おもしろいです。 我摔了好几次，不过挺有意思的。

    4. だいぶ　 [表示程度的副词]
      表示程度相当高，用于 “~ なりました” 等表示变化的句型中，突出强调变化的程度。
      eg: 昨日は　だいぶ　お酒を　飲みました。 昨天喝了不少酒。

    5. 大丈夫 (だいじょうぶ) です
      “没问题”“可以”“没事的”，但可以根据上下文有不同的解释。

    6. 残業 (ざんぎょう)
      在工作时间以外工作，即 “加班”。

    # 第二十课：スミスさんは　ピアノを　弾 (ひ) く　ことが　できます

    # 语法

    # 动词的基本形

    即动词原形，详见 “动词和形容词知识点”。

    类别 ます形 去掉ます 基本形 基本形
    一类动词 書きます かき 書く かく
    急ぎます いそぎ 急ぐ いそぐ
    飛びます とび 飛ぶ とぶ
    読みます よみ 読む よむ
    死にます しに 死ぬ しぬ
    待ちます まち 待つ まつ
    売ります うり 売る うる
    買います かい 買う かう
    話します はなし 話す はなす
    二类动词 食べます たべ 食べる たべる
    見ます み 見る みる
    三类动词 来ます き 来る くる
    します し する する

    # [名 (人)] は　[动 (基本形)] ことが　できます

    “某人会做某事”。“[动 (基本形)] + ことが　できます” 表示能力，在回答别人的询问时，肯定情况下可用 “できます”，否定情况下可用 “できません”。
    森さんは　中華料理 (ちゅうかりょうり) を　作 (つく) る　ことが　できますか。 森先生会做中国菜吗？
    —— はい，できます。 是的，会做。

    # [名] は　[动 (基本形)]　ことです

    当谓语部分的内容是一种动作、行为时使用这个句型。
    eg:
    私は　趣味 (しゅみ) は　切手 (きって) を　集 (あつ) める　ことです。 我的爱好是集邮。
    王さんの　特技 (とくぎ) は　胡弓 (こきゅう) を　弾く　ことです。 小王的特长是拉胡琴。

    “切手 (きって) を　集 (あつ) める　こと” 和 “胡弓 (こきゅう) を　弾く　こと” 相当于名词，即 “动词基本词 + こと” 起到和名词同样的作用，因此也可以用名词或名词短语代替。（类似英语中动名词。）
    小野さんは　車 (くるま) を　運転 (うんてん) する　ことが　できません。 小野不会开车。
    小野さんは　車の運転　が　できません。 小野不会开车。

    # [动 (基本形)] 前に，～

    表示一个动作在另一个动作之前发生时，使用该句型。
    eg：毎日　寝る　前に　シャワーを　浴 (あ) びます。 每天睡觉前淋浴。

    这个句型也可以用 “名词 + の + 前に” 的形式使用。
    会議 (かいぎ)　の　前に，資料 (しりょう) を　コピー　します。　开会之前复印资料。

    # 疑问词 + か

    疑问词 “いつ　どこ　だれ” 等词后续 “か”，表示 “某时 某地 某人” 等意思。
    部屋に　誰か　いますか。 房间里有人吗？ “誰か” 指什么人，某个人
    部屋に　誰が　いますか。 房间里有谁？ “が” 是助词

    “いつか　どこか　だれか　なにか” 等本身并不表示疑问的意思，所以也用于陈述句。
    冬休 (ふゆやす) み，どこかへ　行きたいです。 寒假想去个什么地方。

    # ～よね

    表示提出自己的意见，以征求对方的同意。
    当说话人确信对方和自己的意见、想法完全相同时，只用 “ね”，而说话人对自己的意见、想法没有足够的把握时用 “よね”。
    この　傘 (かさ) は　森さんのですよね。 这把伞是森先生的吧？

    # 表达

    1. 写真を　撮 (と) る　ことが　できます [可能和允许]
      “[动 (基本形)] + ことが　できます” 基本功能是表示可能，但有时根据上下文关系可表示允许。
      寮 (りょう) の　洗濯機 (せんたくき) は　何時まで　使 (つか) う　ことが　できますか。 宿舍的洗衣机可以用到几点？
      ——　８時までです。それからは　使う　ことが　できません。 用到 8 点，那以后就不能用了。

    2. みんなで
      “みんな + で” 组成，这里的で表示动作主体的样态。如：“一人 (ひとり) で（一个人）”“自分 (じぶん) で（自己）”“全員 (ぜんいん) で（全体人员）”。
      eg： みんなで　餃子 (ギョウザ) を　作ります。 大家在一起包饺子。

    3. 趣味 (しゅみ)　和 興味 (きょうみ)
      日语的 “趣味 (しゅみ)” 是 “爱好，喜好” 的意思，表示兴趣的时候，使用 “興味 (きょうみ)”。
      私の　趣味 (しゅみ) は　切手を集める　ことです。 我的爱好是集邮。
      私は　日本の　書道 (しょどう) に　興味 (きょうみ) が　あります。 我对日本的书法感兴趣。

    4. ごちそうします
      “ごちそう” 本身指奢华而高级的菜肴，后续动词 “します” 表示 “请别人吃饭”。
