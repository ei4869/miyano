    本文详细介绍了日语初级上单元三的学习内容，包括第九课至第十二课。第九课主要讲解一类形容词的语法规则，如谓语的使用、否定和过去形式的构成等；第十课则聚焦于二类形容词，包括谓语的一般和过去形式、否定形式等。第十一课进一步探讨了形容词的用法，如表达情感的句型“[名1]は[名2]が[形1]/[形2]です”，以及“[名1]は[名2]が分かります/できます”等表达能力的句型，同时介绍了“や”、“から/だから”、表示频率的副词和询问事由的“どうしてですか”等语法点。第十二课则涉及比较级的表达，如“[名1]は[名2]より[形1]/[形2]です”、“[名1]より[名2]のはうが[形1]/[形2]です”等，以及询问最高级的相关表达。通过这些课程，学习者可以掌握日语中形容词的用法和相关表达方式，为进一步学习打下坚实基础。

    # 第九课：四川料理 (しせんりょうり) は　辛 (から) いです

    # 语法

    # [名] は　[形一] です

    形一是以い结尾的形容词，做谓语时后面加です。

    # [形一] 做谓语时的否定形式

    い变く再加上ないです或ありません。
    eg: 今日は　寒 (さむ) くないです。 今天不冷。

    注：いいです的否定形式是よく　ないです或よく　ありません

    # [形一] 做谓语时的过去形式

    过去形式是：い变かった再加上です。
    过去否定形式是：い变成く　なったです或者く　ありませんでした。
    eg：

    • 旅行 (りょこう) は　楽 (たの) しかったです。（过去） 旅行很愉快。
    • 昨日 (きのう) は　寒 (さむ) くなかったです。 （过去否定） 昨天不冷。
    • 昨日は　寒くありませんでした。 （更正式的表达）

    注：描述过去体验过的事物的性质时，日语需要用过去形式。
    美味 (おい) しいです。 (正在吃的时候)
    美味 (おい) しかったです。 (吃完以后)

    注：いいです的过去形式是よかったです，过去否定形式是よく　ありませんでした。

    # [形一]+[名]

    一类形容词可直接修饰名词。但形容词和名词间不加の。
    eg: 広 (ひろ) い国 (くに)

    # を -> は

    例如：原本应该是コーヒーを　飲みます。但当を前面的名词作为话题时，或用于对比时，变为は。例如谈论喝什么时，可以说コーヒーは 飲みます。

    ###　あまり　[形一 / 动 的否定]
    あまり后面一定接否定形式，表示程度不高。
    eg: この　スープは　あまり　熱 (あつ) く　ないです。 这个汤不太热。

    # 表示程度的副词

    程度比较：とても / 大変 (たいへん) > 少 (すこ) し / ちょっと > あまり (~ ません) > 全然 (ぜんぜん)(~ ません)
    分别是：很，非常 一点儿 不太... 根本不...

    # 表达

    1. 多 (おお) い　和 少 (すく) ない
      不同于其他一类形容词，以上两个不能单独修饰名词。
      eg: 公園に　たくさんの　人が　います。 公园里人很多。

    2. 子供用 (こどもよう)
      "x 用 (よう)" 表示 “x 用的”。如子供用 (こどもよう) 表示 “孩子用的 “。

    3. あら
      女性感到惊讶或疑惑使用的词。

    4. 熱 (あつ) く　ないですか　[否定疑问句]
      表示 “水不烫吗”，在日语中可以用两种方式回答：
      - はい、熱くないです。 是的，不烫。（对不烫这件事，回答是）
      - いいえ、熱くないです。 不，不烫。（对于烫不烫，回答不）

    5. ちょうど　いい
      “正合适”，表示时间或分量合适。

    6. お湯 (ゆ)　和 水 (みず)
      日语中，お湯 (ゆ) 指热水或开水。水 (みず) 仅指凉水。因此可以说 "熱い　お湯（热水）" 和 " 冷 (つめ) たい　水 (みず)"，但不能说" 熱い 水 (みず)"。

    7. 味道的表达方式
      甘 (あま) い (甜的)　(塩 (しお)) 辛 (から) い (咸的)　酸 (す) っぱい (酸的)　辛 (から) い (辣的)　苦 (にが) い (苦的)。
      其中，辛 (から) い有咸和辣两种意思，需要特别区分时，“咸” 可以说成 " 塩 (しお) 辛 (から) い "或" しょっぱい "，" しょっぱい " 只用于口语。
      此外，おいしい（好吃）和まずい（不好吃）表达的是食物或饮料（包括水）放到口中的感觉，因此可以用于修饰水等表达感觉。

    # 第十课：京都 (きょうと) の　紅葉 (もみじ) は　有名 (ゆうめい) です

    # 语法

    # [名] は　[形 2] です / でした

    二类形容词是不以い结尾的形容词。分别是二类形容词做谓语时的一般形式和过去形式。
    eg: この　通 (とお) りは　静 (しず) か　です。 这条街很安静。

    # [名] は　[形 2] では　ありません / ではありませんでした

    分别是二类形容词做谓语时的否定形式和过去形式的否定形式。在口语里，では可以说成じゃ。

    # [名] は　[形 2] な　[名] です

    二类形容词修饰名词词，用な连接。
    eg: 奈良 (なら) は　にぎやか　な　町 (まち) では　ありません。 奈良不是热闹的城市。

    # [名] でした

    名词做谓语时的过去形式是 “名词 + でした”，否定形式是 “名词 + では (じゃ)　ありませんでした”。
    eg: おとといは　曇 (くも) り　じゃ　ありませんでした。 前天不是阴天。

    # どんな　[名]

    询问人或事物的性质时使用，相当于 “什么样的”。
    eg: 張 (ちょう) さんは　どんな　人ですか。 小张是什么样的人？

    注：

    • 何のなんの　有询问内容和材料两种用法。例如：何の　本ですか　—— 自動車 (じどうしゃ) の　本です。 （什么书？—— 关于汽车的书。），这里即询问内容。 再例如：何の　料理 (りょうり) ですか　—— 魚 (さかな) の料理です。 （什么菜啊？—— 鱼。），这里即询问材料。
    • どんな只用于询问性质。往往用形容词回答。例如：どんな　本ですか　—— おもしろい　本です。 （什么样的书？—— 非常有趣的书。）

    # どうですか

    用于询问对方对某状态的意见或感想时，如果询问的是现在或未来的事情则用どうですか，询问的是过去的事情时则用どうでしたか。 另外どうですか有时用于劝诱对方进行某动作。

    # でも　和 そして

    でも表示转折关系，一般用于口语，书面语用しかし。
    そして表示并列关系。

    # 表达

    1. 綺麗 (きれい) です
      注意虽然以い结尾，但其实是二类形容词，きれい的い是綺麗汉字读音的一部分。
      eg: きれいな　服 (ふく) 漂亮的衣服

    2. おとといは　雨 (あめ) でした
      表示 “下雨”“下雪” 等自然现象时，日语里通常用 “雨 (あめ) です”“雪 (ゆき) です”。

    3. あれえ
      感到吃惊和意外情不自禁发出的声音。

    4. ところで
      转换话题时使用。
      eg: ああ、そうですか。ところで、この　美術館 (びじゅつかん) には　どんな作品 (さくひん) が　ありますか。 噢，是吗。哎，请问这个美术馆里都有什么作品？

    5. 世界中 (せかいじゅう)
      "[场所]+ 中 (じゅう)"表示其场所范围内的全部。"[时间 / 期间]+ 中 (じゅう)"时则用" 一 (いち / ひと)~ 中 (じゅう)" 的形式，表示 “在某个期间一直”。

    6. 形容词 + の
      一般用于避免重复使用词语，用の指代了前面谈话的内容。如：おもしろいのが　ありますよ（还有很有趣的），这里的有趣指的是前面谈到过的雕刻等话题。
      对于二类形容词，要在の前加な，如 “簡単 (かんたん) なの（简单的）”。

    # 第十一课：小野さんは　歌 (うた) が　好きです

    # 语法

    # [名 1] は　[名 2] が　[形 1]/[形 2] です

    形容词 1 和形容词 2 为表达情感的形容词。
    名词 1 表示情感的主题，名词 2 表示情感的对象。
    eg: 林 (はやし) さんは　犬 (いぬ)　が　怖 (こわ) いです。 林先生害怕狗。

    # [名 1] は　[名 2] が　分 (わ) かります / できます

    分かります（明白）　できます（会）　上手 (じょうず) です（擅长）　下手 (へた) です（不擅长（客观上）） 苦手 (にがて) です（不擅长（主观情感上）） 等表达能力的词语也可用该句型。
    eg: 私は　コンピュータ　が　できません。　 我不会用电脑。

    # [名] や　[名]

    や也可用于连接名词，但と用于列举所有的项目，や只用于列举许多项目中的两项，相当于 “呀什么的”，常与など（等等）呼应使用。
    eg: 私は　お店 (みせ) で　野菜 (やさい) や　果物 (くだもの) などを　買いました。 我在店里买了蔬菜呀水果等等。

    # ~ から　和 だから [原因、理由]

    から需要接在表示原因、理由小句的句尾。一般形 1 / 动词 + から。
    eg: 森さんは　お酒 (さけ) が　好きですから、毎日　飲みます。 森先生喜欢喝酒，每天都喝。

    ですから更有礼貌。

    だから是表示原因、理由的连词，相当于 “所以”，一般用在结果的句子的前面。礼貌的说法是ですから。
    eg: 明日 (あした) は　休みです。ですから、子供と　動物園 (どうぶつえん) へ　行きます。 明天休息，所以我跟孩子去动物园。

    # 表示频率的副词

    いつも　よく　時 (ときどき)々　たまに　あまり　全然 (ぜんぜん) 等为表示频率的副词。あまり、全然后面接否定形式。

    # どうしてですか

    询问事由时使用，相当于 “为什么”。回答时在谓语后面加から。
    eg:
    - ぼくは　来月 (らいげつ)　また　ここへ　来きますよ。 我下个月还来这里。
    - どうしてですか。 为什么？
    - ここで　友達の　結婚式 (けっこんしき) が　ありますから。 因为朋友要在这里举行结婚典礼。

    # 表达

    1. 僕 (ぼく)
      男性对关系比较亲密的人自称时使用的第一人称代词。

    2. ホテルで　写真展 (しゃしんてん) が　あります
      某处发生某事，某事发生的场所用で。 区分之前：[场所] に　[物] が　あります。

    3. いかがですか
      是どうですか的礼貌表达，对上级或长辈使用。表示 “~ 怎么样”。
      eg:
      - コーヒーは　いかがですか。
      接受时：どうも ありがとう　ございます　いただきます。
      拒绝时：今は　結構 (けっこう) です。（现在不用了。）

    4. そうですね
      可以表示同意对方提议，也可用在疑问句后面，表示说话人在思考接下来要说的内容。

    5. 迷 (まよ) います
      用于难以立即做出选择的场合。表示犹豫。也可以表示 “迷路”。

    6. 気 (き) に　入 (い) りました
      相当于 “喜欢”，但不表示自己原有的爱好，仅用于第一次见到时感到中意，对象用が，且只用于朋友或熟人之间。
      表达持续性的爱好用好きです。

    # 第十二课：李さんは　森さんより　若 (わか) いです

    # 语法

    # [名 1] は　[名 2] より　[形 1]/[形 2] です

    “名 1 比名 2 更～”。
    eg: 上海 (シャンハイ) は　北京 (ペキン) より　にぎやかですか。 上海比北京热闹吗？

    程度差异较大时用副词 “ずっと” 强调。
    eg: 中国は　日本より　ずっと　広 (ひろ) いです。 中国比日本辽阔得多。

    # [名 1] より　[名 2] の　はうが　[形 1]/[形 2] です

    “与名 1 相比，名 2 更～”。
    eg: 大阪 (おおさか) より　東京 (とうきょう) の　ほうが　にぎやかですか。 与大阪相比，东京更热闹吗？

    # [名 1] は [名 2] ほど [形 1] く ないです /[形 2] では　ありません

    “名 1 没有名 2 那么～”。
    eg: 東京の　冬 (ふゆ) は　北京の　冬ほど　寒く　ないです。 东京的冬天没有北京的冬天那么冷。（形 1）
    神戸 (こうべ) は　大阪　ほど　にぎやか　ではありません。 神户没有大阪那么繁华。 （形 2）

    # [名 1] の　中 (なか) で　[名 2] が　一番 (いちばん)　[形 1]/[形 2] です

    “在名 1 中，名 2 是最～的”。
    eg: 日本料理 (りょうり) の　中で　寿司 (すし) が　いちばん　おいしいです。 日式饭菜中，寿司最好吃。

    注：如果 名词 1 表示场所或时间，则后面不加 “の　中 (なか)”，而使用 “[名 1 (场所、时间)] で　いちばん　[形 1 / 形 2] [名 2] は　[名 3]　です”。　　表示：在 场所 / 时间 ，最～的名 2 是名 3。
    eg: 日本で　いちばん　高 (たか) い　山 (やま) は　富士山 (ふじさん) です。 日本最高的山是富士山。

    注：日语中的 “いちばん” 只能用于 “~ の　中で”　“~ で” 等明确指出比较对象和范围的情况。

    # [名 1] と　[名 2] と　どちらが　[形 1]/[形 2] ですか

    询问两种事物当中哪一种更怎么样。“どちら” 是两者二选一时使用的疑问词，回答时使用 “の　ほうが...”。如果两者程度相当，则使用 “どちらも　~ です”。“どちら” 在口语中可说成 “どっち”。
    eg:
    - 日本語と　英語と　どちらが　難 (むずか) しいですか。 日语和英语，哪个难学？
    —— 日本語の　ほうが　難しいです。　 日语难学。

    询问的范围为两种事物时，无论是人是物还是时间，都可用 “どちらが”。但询问的范围为三种以上的事物时，不能用 “どちら”。

    # どの　[名] / いつ / どれ / だれ / 何 (なに) 　が　いちばん　[形 1]/[形 2] ですか

    询问三个以上事物中哪个最～时，根据询问对象不同需要区分：是人则使用 “だれが”，是物则使用 “どの　~ が” “どれが” 或者 “何が”，是时间则使用 “いつが”。
    eg：
    - この　中で　どの　料理　が　いちばん　おいしいですか。 在这些菜当中，哪个菜最好吃？
    —— この　料理　が　いちばん　おいしいです。 这个菜最好吃。

    - 季節 (きせつ) の　中で　いつが　いちばん　好きですか。 四个季节当中，你最喜欢哪一个？
    ——　私は　春 (はる)　が　いちばん　好きです。 我喜欢春天。

    被选择的事物已见于上下文或存在于眼前时，使用 “どの　～が” 或 “どれが”，反之则使用 “何が”。
    eg: リンゴと　ナシと　バナナの　中で　どれが　いちばん　好きですか。 苹果、梨和香蕉当中你最喜欢什么？

    # 表达

    1. 最近 (さいきん)
      表示从较近的过去到现在为止的一段时间。由于是说话人自己的判断，所以所指时间长短因人而异。且 “最近 (さいきん)” 不能表示将来的时间。
      eg: 最近　十年間 (じゅうねんかん) は　元気 (げんき) でした。 最近 10 年身体都很好。

    2. ~ か　いいです
      根据上下文判断意思，例如：
      - 箱根 (はこね) の　お土産 (みやげ) は　何が　いいですか。 箱根的礼品什么好？
      —— 寄木細工 (よせぎざい) が　いいです。 木片儿拼花工艺品比较好。 （表示选择）

    - 私は　肉 (にく) より　魚 (さかな)　ほうが　いいです。 比起肉来我更喜欢吃鱼。 （表示喜欢）

    1. やっぱり
      是 “やはり” 较随便的说法，表示 “正如自己预料的那样”。也可表示 “预测了各种各样的结局，结果还是～” 的意思，表示经过一番考虑确定了某个想法。

    2. ~ が... [说半句话时]
      话不说完，用が结尾，通常在遇到难以启齿的事或拒绝别人的请求时这样做，给对方委婉、含蓄的印象。
      eg: あのう、それは　私の　席 (せき) ですが... 对不起，那是我的座位......
      也可用于补充和前面所说内容有出入的情况。
      eg: どちらが　おいしいですか。　——　どちらも　おいしいですよ。私は　ジャスミン茶 (ちゃ) が　好きですが...。
      (哪种好喝？—— 哪种都好喝，不过我更喜欢喝茉莉花茶。)
