    简明摘要： 本文介绍了日语初级上单元三的学习内容，涉及第九课和第十课。第九课主要讲解一类形容词的语法规则，包括谓语的使用、否定和过去形式的构成，以及“形一+名”、“を→は”等语法点。同时，介绍了一些常用词汇和表达方式，如“あまり”、程度副词、“多い”和“少ない”等。第十课则聚焦于二类形容词，包括谓语的一般和过去形式、否定形式、修饰名词的用法，以及“名でした”等语法点。此外，还涉及了一些表达方式，如“きれい”、“おととい”等。通过这些课程，学习者可以掌握日语中形容词的用法和相关表达方式。

    # 第九课：四川料理 (しせんりょうり) は　辛 (から) いです

    # 语法

    # [名] は　[形一] です

    形一是以い结尾的形容词，做谓语时后面加です。

    # [形一] 做谓语时的否定形式

    い变く再加上ないです或ありません。
    eg: 今日は　寒 (さむ) くないです。 今天不冷。

    注：いいです的否定形式是よく　ないです或よく　ありません

    # [形一] 做谓语时的过去形式

    过去形式是：い变かった再加上です。
    过去否定形式是：い变成く　なったです或者く　ありませんでした。
    eg：

    • 旅行 (りょこう) は　楽 (たの) しかったです。（过去） 旅行很愉快。
    • 昨日 (きのう) は　寒 (さむ) くなかったです。 （过去否定） 昨天不冷。
    • 昨日は　寒くありませんでした。 （更正式的表达）

    注：描述过去体验过的事物的性质时，日语需要用过去形式。
    美味 (おい) しいです。 (正在吃的时候)
    美味 (おい) しかったです。 (吃完以后)

    注：いいです的过去形式是よかったです，过去否定形式是よく　ありませんでした。

    # [形一]+[名]

    一类形容词可直接修饰名词。但形容词和名词间不加の。
    eg: 広 (ひろ) い国 (くに)

    # を -> は

    例如：原本应该是コーヒーを　飲みます。但当を前面的名词作为话题时，或用于对比时，变为は。例如谈论喝什么时，可以说コーヒーは 飲みます。

    ###　あまり　[形一 / 动 的否定]
    あまり后面一定接否定形式，表示程度不高。
    eg: この　スープは　あまり　熱 (あつ) く　ないです。 这个汤不太热。

    # 表示程度的副词

    程度比较：とても / 大変 (たいへん) > 少 (すこ) し / ちょっと > あまり (~ ません) > 全然 (ぜんぜん)(~ ません)
    分别是：很，非常 一点儿 不太... 根本不...

    # 表达

    1. 多 (おお) い　和 少 (すく) ない
      不同于其他一类形容词，以上两个不能单独修饰名词。
      eg: 公園に　たくさんの　人が　います。 公园里人很多。

    2. 子供用 (こどもよう)
      "x 用 (よう)" 表示 “x 用的”。如子供用 (こどもよう) 表示 “孩子用的 “。

    3. あら
      女性感到惊讶或疑惑使用的词。

    4. 熱 (あつ) く　ないですか　[否定疑问句]
      表示 “水不烫吗”，在日语中可以用两种方式回答：
      - はい、熱くないです。 是的，不烫。（对不烫这件事，回答是）
      - いいえ、熱くないです。 不，不烫。（对于烫不烫，回答不）

    5. ちょうど　いい
      “正合适”，表示时间或分量合适。

    6. お湯 (ゆ)　和 水 (みず)
      日语中，お湯 (ゆ) 指热水或开水。水 (みず) 仅指凉水。因此可以说 "熱い　お湯（热水）" 和 " 冷 (つめ) たい　水 (みず)"，但不能说" 熱い 水 (みず)"。

    7. 味道的表达方式
      甘 (あま) い (甜的)　(塩 (しお)) 辛 (から) い (咸的)　酸 (す) っぱい (酸的)　辛 (から) い (辣的)　苦 (にが) い (苦的)。
      其中，辛 (から) い有咸和辣两种意思，需要特别区分时，“咸” 可以说成 " 塩 (しお) 辛 (から) い "或" しょっぱい "，" しょっぱい " 只用于口语。
      此外，おいしい（好吃）和まずい（不好吃）表达的是食物或饮料（包括水）放到口中的感觉，因此可以用于修饰水等表达感觉。

    # 第十课：京都 (きょうと) の　紅葉 (もみじ) は　有名 (ゆうめい) です

    # 语法

    # [名] は　[形 2] です / でした

    二类形容词是不以い结尾的形容词。分别是二类形容词做谓语时的一般形式和过去形式。
    eg: この　通 (とお) りは　静 (しず) か　です。 这条街很安静。

    # [名] は　[形 2] では　ありません / ではありませんでした

    分别是二类形容词做谓语时的否定形式和过去形式的否定形式。在口语里，では可以说成じゃ。

    # [名] は　[形 2] な　[名] です

    二类形容词修饰名词词，用な连接。
    eg: 奈良 (なら) は　にぎやか　な　町 (まち) では　ありません。 奈良不是热闹的城市。

    # [名] でした

    名词做谓语时的过去形式是 “名词 + でした”，否定形式是 “名词 + では (じゃ)　ありませんでした”。
    eg: おとといは　曇 (くも) り　じゃ　ありませんでした。 前天不是阴天。

    # どんな　[名]

    询问人或事物的性质时使用，相当于 “什么样的”。
    eg: 張 (ちょう) さんは　どんな　人ですか。 小张是什么样的人？

    注：

    • 何のなんの　有询问内容和材料两种用法。例如：何の　本ですか　—— 自動車 (じどうしゃ) の　本です。 （什么书？—— 关于汽车的书。），这里即询问内容。 再例如：何の　料理 (りょうり) ですか　—— 魚 (さかな) の料理です。 （什么菜啊？—— 鱼。），这里即询问材料。
    • どんな只用于询问性质。往往用形容词回答。例如：どんな　本ですか　—— おもしろい　本です。 （什么样的书？—— 非常有趣的书。）

    # どうですか

    用于询问对方对某状态的意见或感想时，如果询问的是现在或未来的事情则用どうですか，询问的是过去的事情时则用どうでしたか。 另外どうですか有时用于劝诱对方进行某动作。

    # でも　和 そして

    でも表示转折关系，一般用于口语，书面语用しかし。
    そして表示并列关系。

    # 表达

    1. 綺麗 (きれい) です
      注意虽然以い结尾，但其实是二类形容词，きれい的い是綺麗汉字读音的一部分。
      eg: きれいな　服 (ふく) 漂亮的衣服

    2. おとといは　雨 (あめ) でした
      表示 “下雨”“下雪” 等自然现象时，日语里通常用 “雨 (あめ) です”“雪 (ゆき) です”。

    3. あれえ
      感到吃惊和意外情不自禁发出的声音。

    4. ところで
      转换话题时使用。
      eg: ああ、そうですか。ところで、この　美術館 (びじゅつかん) には　どんな作品 (さくひん) が　ありますか。 噢，是吗。哎，请问这个美术馆里都有什么作品？

    5. 世界中 (せかいじゅう)
      "[场所]+ 中 (じゅう)"表示其场所范围内的全部。"[时间 / 期间]+ 中 (じゅう)"时则用" 一 (いち / ひと)~ 中 (じゅう)" 的形式，表示 “在某个期间一直”。

    6. 形容词 + の
      一般用于避免重复使用词语，用の指代了前面谈话的内容。如：おもしろいのが　ありますよ（还有很有趣的），这里的有趣指的是前面谈到过的雕刻等话题。
      对于二类形容词，要在の前加な，如 “簡単 (かんたん) なの（简单的）”。
