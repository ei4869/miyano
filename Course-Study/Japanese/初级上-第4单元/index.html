    # 第十三课：机 (つくえ) の　上に　本が　三冊 (さつ)　あります

    # 语法

    # [名 (数量)] + [动]

    数量词用在动词前面，量词因所修饰的事物种类而不同。

    注：询问数量时，在 “何 (なん)” 的后面加量词，如 “何個 (なんこ)（多少个）”“何本 (なんぼん)（多少根）”“何冊 (なんさつ)（多少本）”。

    量词 使用对象
    ~ 人 (にん) 人
    ~ 台 (だい) 机械或车辆：车、机动车、电脑等
    ~ 枚 (まい) 薄平物品：纸、盘子、邮票、衬衫等
    ~ 冊 (さつ) 书、笔记本等
    ~ 歳 (さい) 年龄
    ~ 回 (かい) 次数
    ~ 着 (ちゃく) 衣服等
    ~ 個 (こ) 立体的物品：鸡蛋、饭团子、球等
    空容器或器皿：箱子、盒子、空罐、玻璃杯等
    概念性的、抽象的事物：密码等
    ~ 本 (ぼん) 细长物品：瓶、铅笔、伞、香蕉等
    ~ 杯 (はい) 容器内的饮料：茶、咖啡、啤酒等
    ~ 匹 (ひき) 小动物：狗、猫、昆虫、鱼等
    ~ 頭 (とう) 大动物：牛、马、象等
    ~ 羽 (わ) 鸟、兔
    ~ 番 (ばん) 顺序
    ~ 足 (そく) 成对物品：鞋、袜子等
    ~ つ 1-9 岁的年龄
    立体的物品：橘子、帽子等
    抽象的事物：谜、选项等
    ”~ つ “不用于 10 以上的数字

    # [名 (时间)] + [动]

    表示时间数量的词语和动词一起使用时，说明动作、状态的持续时间。这时表示时间数量的词语后面不能加に。
    eg: 李さんは　毎日　七時間 (しちじかん)　働 (はたら) きます。 小李每天工作 7 小时。

    # [名 (时间)] に　[名 (次数)] + [动]

    表示在一定时间内进行若干次动作时，使用该句型。
    eg: この　花 (はな) は　二年 (にねん) に　一度 (いちど)　咲 (さ) きます。 这种花每两年开一次。

    # [名 (场所)] へ　[动] に　行 (い) きます / 来 (き) ます

    “去某地做某事”。 表示移动行为的目的时使用，句型中表示场所的名词为移动行为的目的地；动词为 “ます形” 去掉 “ます” 的形式，表示移动行为的目的。
    eg: 李さんは　私の家　へ　遊 (あそ) び　に　来ました。 小李来我家玩了。 （遊びます　-> 遊び）

    # [名 (数量)] + で

    用于不称重量而以数个的方式售物。
    eg: この　ケーキは　三個 (さんこ) で　五百円 (ごひゃくえん) です。 这种蛋糕 3 个 500 日元。

    不过数量是 1 个时，不加で。
    eg: この　ケーキは　一個 (いっこ)　二百円です。 这种蛋糕 1 个 200 日元。

    # 表达

    (2) 週に (しゅう) に　二回 (かい)
    “週に (しゅう) に　二回 (かい)“ 的完整表达时 “一週間 (いっしゅうかん) に　二回 “，像这样以一定期间为基准计算频率时，常常省略数字”1 “和” 間 (かん)“，而使用” 週 (しゅう) に　~ 回 (かい)“的形式。“一日 (いちにち) に” 变成 “日 (ひ) に”

    (3) 生 (なま) ビールを　三 (みっ) つ 来三杯生啤
    属于省去量词直接使用 “一 (ひと) つ　二 (ふた) つ” 等方式计数的情况。

    1. くらい　/ ぐらい
      在表示数量的词后加 “くらい”“ぐらい” 表示大概的数量。有时还与副词 “大体 (だいたい)” 同时使用。
      eg:
      この　パソコンは　いくらでしたか。 这台电脑多少钱买的？
      —— だいたい　二十万円 (にじゅうまんえん) ぐらいです。 大约 20 万日元。

    2. どのぐらい　かかりますか
      “かかります” 用于时间和钱，相当于 “需要，花费”。（多少时间，多少钱）
      eg:
      家から　会社まで　どのぐらい　かかりますか。 从你家到公司需要多少时间。
      —— 電車で　一時間ぐらい　かかります。 乘电车需要 1 小时左右。

    3. 飲みに行きます
      不指明说 “お茶を”“ジュースを”，而只说 飲みに行きます ，则指 “去喝酒”。

    4. とりあえず
      从具有可能性的几种动作或事项中，暂且选择一种先做。 “暂且先...”

    # 第十四课：昨日　デパートへ　行って，買 (か) い物 (もの) しました

    # 语法

    # [动] て　[动] 相继发生

    表达两个以上的动作按照时间相继发生时，可用 “动词て形 + 动词” 表示。 即用一句话表达连续的动作。
    eg: 家へ　帰って，宿題 (しゅくだい) を　します。 回家，（然后）写作业。

    # [动] て　から　[动] 相继发生

    也可以表述两个以上动作按照时间顺序相继发生。，和上一点区别是，“てから” 不能再一个句子中反复使用两次以上。 即上一点可以多个て形连用，这个句型只能使用一次て形。

    eg:
    図書館へ　行って　本を　借 (か) りて，家へ　帰ります。 去图书馆借书，（然后）回家。
    昼 (ひる) ご飯を　食べてから　出 (で) かけます。 吃了午饭后出门。

    # [动] て ください

    请求某人做某事。
    eg: ここに　住所 (じゅうしょ) と　名前 (なまえ) を　書 (か) いて　ください。 请在这里填写住址和姓名。

    # [名 (场所)] を　[动] 经过 离开

    经过的场所用助词を表示，常用动词有 “通 (とお) ります”“渡 (わた) ります”“過 (す) ぎます” 等。此外，を与动词 “出 (で) ます”“卒業 (そつぎょう) します” 等搭配使用时，表示离开的场所。
    eg:
    この　パスは　駅前 (えきまえ)　を　通ります。 这趟公共汽车经过车站一带。

    駅前 (えきまえ) 指车站周围的整个区域。明确表示 “车站前” 时使用 “駅 (えき) の　前”

    私は　来年　大学 (だいがく)　を　卒業 (そつぎょう) します。 我明年大学毕业。

    # 表达

    1. 句子的连接方式
      “~ て” 是一个连续相继发生动作的形式，但连接太多会使人难以理解，所以一般只连接两项或三项，需要连接更多动作时，使用 “それから”。
      eg: 図書館へ　行って，勉強を　します。それから　家へ　帰って，手紙を　書きます。 去图书馆学习。然后回家写信。

    2. 買 (か) い物 (もの) します [三类动词]
      一般 “します” 前接汉字或外来词可以构成三类动词。
      这种形式也可以说成 “名词 + を + します”（を可不加。）
      eg:
      出張 (しゅっちょう) します　出差；仕事 (しごと) します 工作 （汉字词）
      ジョギングします 慢跑；ファックスします 发传真 （外来词）

    3. なかなか 表示程度的副词
      一般用于上级对下级。后续接肯定表达时，表示从自己个人角度看程度属于上乘。
      eg: 李さんの　日本語は　なかなか　上手 (じょうず) ですね。 小李的日语相当棒！
      该词含有实际情况比自己预想的程度要高的含义，一般用于说话人预想程度较低时。

    4. ~ て　くださいませんか
      加 “ませんか” 是更为礼貌的形式，请求某人做某事。

    5. もう
      前面学过 “已经” 意思，表示事情已经结束，但还有 “马上，就要” 的意思，此时句尾不是用 “ました”，而是 “ます”。
      eg: 李さん，もう　会社を　出 (で) ますか。 小李，这就走吗？

    6. すみませんが，~ 搭话
      请求别人做某事时，句子前面加 “すみません” 或者 “すみませんが”，表示因占用对方时间而感到过意不去，表示 “麻烦你”“不好意思”。一般家人或亲切的朋友间不怎么使用。

    7. そうして　ください
      “请那样做 “。这里的” そう “承接了上面的内容，用于指代前面的内容。 其中涉及变形：そうする　-> そうして
      eg:
      速達 (そくたつ) で　出 (だ) しますか。 寄快件吗？
      —— ええ，そうして　ください。 对，寄快件。

    8. お金 (かね) を　下 (お) ろします
      “下 (お) ろします” 表示将上面的东西取下来，这里意为 “把存在银行或邮局的钱从账户中取出来”。

    # 第十五课：小野さんは　今　新聞 (しんぶん) を　読 (よ) んで　います

    # 语法

    # [动] て　います 正在进行

    表示动作或变化正在进行时。“正在做某事”。
    eg: 森さんは　今　仕事を　しています。 森先生正在工作。

    # [动] ても　いいです

    表示许可时，“可以做某事”。[动] ても　いいですか。　“可以做某事吗？”
    eg: 家で　仕事を　しても　いいです。 在家里工作也可以。
    家に　帰っても　いいですか。 可以回家吗？
    —— はい，帰っても　いいです。 是的，可以回家。

    其中，” はい，帰っても　いいです “可以省略为” はい，いいです。“
    表示说话人的许可时，使用 “いですよ”，语气更柔和，读升调，但不用于尊长。

    # [动] ては　いけません

    表示禁止时，“禁止做某事”。
    eg:
    この　部屋に　入 (はい) っては　いけませんか。 不能进这个屋子吗？
    —— はい，入っては　いけません。 是的，不能。
    —— いいえ，入っても　いいです。 不，可以进。
    对尊长一般不用 “~ ては　いけません” 而使用 “~ ないで　ください”。

    # [名 (附着点)] に　[动]

    表示人或物体的附着点，即人或物体停留在交通工具或椅子上等时，附着点用助词 “に” 表示。
    eg: 李さんは　電車に　乗 (の) りました。 小李上了电车。
    この　いす　に　座 (すわ) って　ください。 请坐在这把椅子上。

    不能用を连接！

    # [名 (目的地)] に　[动]

    移动行为的目的地既可以用へ表示，也可以用に表示。
    eg: 李さんは　病院 (びょういん)　に　行きます。 小李去医院。

    目前为止：
    “に” 的用法包括：

    1. 表示存在的场所，例如 “公園にベンチがあります”（公园里有长椅）。
    2. 表示时间，例如 “5 時に帰ります”（5 点回家）。
    3. 表示动作的对象，例如 “友達に会います”（和朋友见面）。
    4. 表示移动行为的目的，例如 “本を買いに行きます”（去买书）。
    5. 表示附着点，例如 “車に乗ります”（上车）。

    “に” 和 “へ” 只有在表示 “行きます”（去）、“来ます”（来）、“帰ります”（回来）等典型的移动动词的目的地时才可以通用。

    # 表达

    1. 请求许可时的回答方式
      (1) 肯定的场合
      对 “~ ても　いいですか” 进行肯定回答时，如果根据自己的心情或判断表示许可时，可以回答 “どうぞ (可以)”“かまいません (没关系)” 等。
      如果不是根据自己的判断，而是基于社会惯例或公共准则进行回答时，可以说 “いいです (可以)”“大丈夫 (だいじょうぶ) です (没关系)” 等。

    (2) 否定的场合
    对 “~ ても　いいですか” 进行否定回答时，如果根据自己的心情或判断表示拒绝时，可以回答 “いえ，ちょっと...(不，有点...)” 婉言拒绝，或者使用 “すみません，~ から (对不起，因为...)” 这种陈述理由的拒绝方式。
    如果是根据法律、规章、社会惯例做出判断时，可以回答 “いけません (不可以)”“だめです (不行)” 等。

    1. もちろんです
      “当然可以”。

    2. 無理 (むり) を　しては　いけませんよ
      “不能勉强”，由 “無理を　します (勉强)” 转换而成。“無理を　します (勉强)” 指行为超过了体力或经济能力的界限，可能给身体或事态造成负面效果。

    3. ゆっくり
      表示动作速度慢，还可以表示 “好好地” 的意思。
      eg: じゅあ，病院へ　行ってから，ゆっくり　休 (やす) んで　ください。 那么，先去医院，然后好好休息一下。

    4. 薬 (くすり) を　出 (だ) します
      医生在医院或诊所给患者开药方。不能用 “売 (う) ります (卖药)” 或者 “あげます (给药)”，只能 “出します”。

    5. お大事 (だいじ) に 寒暄语
      是 “どうぞ　お大事に　してください (请多保重)” 的省略形式，用于对生病或受伤的人表示关心。不能对身体健康的人使用。

    # 第十六课：ホテルの　部屋は　広 (ひろ) くて　明 (あか) るいです

    # 语法

    # [形 1] て　[形 1] 并列

    表示 “又～又～”，两个以上一类形容词并列使用时，使用一类形容词的て形，形 1 变て形是：い -> くて。
    eg:
    明るい　-> 明るくて
    新 (あたら) しい　-> 新しくて

    この　本は　安 (やす) くて　おもしろいです。 这本书又便宜又有意思。

    “[形 1] て　[形 1] ” 也可以用于修饰名词。
    eg: 軽 (かる) くて　暖 (あたた) かい　コートを　買いました。 买了一件轻巧又暖和的大衣。

    # [形 2] で　[形 2] 并列

    两个以上二类形容词并列使用时，使用二类形容词的て形，形 2 变て形是：直接加で。形 1 和形 2 也可以连接使用。
    eg:
    簡単 (かんたん)　-> 簡単で
    親切 (しんせつ)　-> 親切で

    eg: この　公園は　静 (しず) かで　広いです。 这个公园安静而宽敞。

    # [名 1] で　[名 2]　并列

    两个以上的名词并列使用时，用 “[名 1] で　[名 2]” 的形式，名词变て形是：直接加で。
    eg: この　機械 (きかい) は　日中電気 (にっちゅうでんき) の　製品 (せいひん) で，中国製 (ちゅうごくせい) です。 这台机器是日中电器公司的产品，是中国制造的。

    # [动] て　います 结果状态

    “[动] て　います” 除了表示正在进行，还可表示动作结束后留下的结果状态。
    eg: 森さんは　車 (くるま) を　持 (も) って　います。 森先生有辆车。

    对用表示结果状态的 “~ て　います” 提问的句子做否定回答时，使用 “~ て　いません”。但是，对 “知 (し) っていますか” 做否定回答时必须使用 “知 (し) りません（不知道，不认识）”

    # [小句] が，[小句] 转折

    表示两个小句的转折关系时，用助词 “が”，相当于 “但是，可是”。
    eg: この　レストランは　おいしいですが，ちょっと　高 (たか) いです。 这家餐厅的饭菜好吃，但有点贵。

    # まだ + [动 (否定)]

    表示还没达到所询问的内容的程度，相当于 “还没有”。
    eg:
    何年ぐらい　日本語を　勉強しましたか。 学了几年日语？
    —— 二年　勉強しました。でも，まだ　あまり　できません。 学了两年，但还不太会。

    此外，对用 “もう” 提问的句子做否定回答时，使用 “まだ　~ て　いません” 的表达形式。
    もう　結婚していますか。 —— いいえ，まだ　結婚していません。

    # 表达

    1. 背 (せ) が　高 (たか) くて　脚 (あし) が　長 (なが) くて 相貌身材

    顔 (かお) 脸 大きい：大；小さい：小
    背 (せ) 个子 高い：高；低 (ひく) い：矮
    1. 入場料 (にゅうじょうりょう) 会費 (かいひ)
    费用 示例
    ～費 (ひ) 会費 (かいひ) 会费、光熱費 (こうねつひ) 照明、供热、供暖费、交通費 (こうつうひ) 交通费、食費 (しょくひ) 伙食费
    ～料 (りょう) 入場料 (にゅうじょうりょう) 门票费、使用料 (しようりょう) 使用费、送料 (そうりょう) 运费、手数料 (てすうりょう) 手续费
    ～代 (だい) 電気代 (でんきだい) 电费、水道代 (すいどうだい) 水费、タクシー代 (だい) 出租车费、食事代 (しょくじだい) 餐费

    1. ID(アイディー) カードは　ありますか （ィ li）
      前面学过的 “あります” 包括：表示存在、表示在某处发生某事，这里表示所有，如果所有的对象是人，则用 “います”。
      eg:
      私は　車が　二台 (だい)　あります。 我有两辆车。
      森さんは　弟が　二人 (ふたり)　います。 森先生有两个弟弟。

    2. ちゃんと
      副词，表示 “没错” 或 “处于正常状态”，只用于口语。
      eg: 皆さん，机の　上を　ちゃんと　片付 (かたづ) けて くださいね。 请大家把桌上的东西收拾好。（づ du）

    3. ほら
      “你看”，用于提醒别人注意。

    4. ずいぶん
      表示程度高，内容上无论是好是坏，用于事态的程度大幅超过说话人自身或者一般性的评判标准。
      eg: 李さん，日本語が　ずいぶん　上手 (じょうず) ですね。 小李，你的日语真棒啊。

    5. 派手 (はで)
      形容过分鲜艳的色彩或奇特的形状，“耀眼，花哨”，反义词是 “地味 (じみ)（朴素，质朴）”

    6. 嫌 (きら) いじゃ　ありません
      表示并不十分喜欢，但也不讨厌，形式是否定的，但语义上是正面、肯定的评价。
