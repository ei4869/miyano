    本文是日语初级下单元七的学习资料，涵盖两课内容。第二十五课讲解动词修饰名词的语法结构，如动词简体形+名、小句简体形+名，以及形容词修饰名词的规则。第二十六课介绍动词小句名词化的结构，如小句简体形+の+は/を+形/动，以及可能发生的表达方式“かもしれません”。这些内容有助于提高日语学习者的语法应用能力。文章还介绍了日语中表达可能性、原因、结果、习惯性动作和互动行为的语法和表达方式，如“もしかしたら”表达可能性，“それで”表示原因，“つい”表达无意后果，“あいさつ回り”指拜访新环境，“ながら”表示同时进行两个动作，“でしょう”用于确认信息，“ています”表示反复或习惯，“で”表示原因，“と会います”表示互动行为等。此外，还介绍了“アルバイト”、“学校に通っています”、“大勢”、“お年寄り”、“そう而言えば”和“へえ”等表达。通过这些语法和表达，可以更自然地进行日语交流。文章主要介绍了日语中表达请求、动作归属和地点的几种句型，如“动词て形+くれます”表示别人为说话人或其一方的人做事，用于请求亲密关系的人帮忙更礼貌；“～てあげます”含有赐恩意味，对尊长不宜使用；“それに”表累加关系，多用于口语；“～先”表示动作的到达地点或归属；“どの辺”询问大致场所；“どうしたんですか”询问某物的购入方式。这些句型有助于学习者更准确地表达请求、动作归属和地点。

    # 第二十五课：これは明日会議 (かいぎ) で使 (つか) う資料 (しりょう) です

    # 语法

    # [动词 (简体形)] + [名]、[小句 (动词简体形)] + [名]

    动词或由动词作谓语的小句（简称动词小句）修饰名词时，用动词的简体形。 动词修饰名词
    具体说，表示某种习惯或将要发生的动作时用动词的基本形，表示动作已经完成时用动词的 “た形”。

    あれは　倒産 (とうさん) した　会社　です。 那是一家倒闭的公司。

    注：名词前不用敬体形。

    さっき森さんが　座 (すわ) った　場所 (ばしょ)。 森先生刚才坐过的地方。 （た形）
    森さんが　知らない　ニュース。 森先生不知道的新闻。 （未然形）

    注：名词前不能加 “の”。

    森さんが　座る場所。 森先生要做的地方。
    あれは　昨夜 (ゆうべ)　泊 (と) まった　ホテルです。 那是昨天晚上住的宾馆。

    # [小句 (动词简体形)] + [名] は　[名]/[形] です

    动词小句修饰名词而形成的名词短语常常用来作主语。

    私が明日乗 (の) る飛行機 (ひこうき) は　中国航空 (こうくう) です。 我明天乘坐的飞机是中国航空公司的。
    小句中的主语要用 “が” 连接谓语。
    小野さんが作る料理　は　おいしい　です。 小野女士做的菜很好吃。

    即名词前面跟一个长的定语用于修饰名词。如：昨日父 (ちち) がデパートで買 (か) った日本製 (せい) のカメラはとても安やすかったです。 昨天爸爸在百货商店买的日本造的照相机很便宜。

    # [小句 (动词简体形)] + [名] を / に / から [动] ます

    动词小句修饰名词而形成的名词短语也可以用作主语以外的其他句子成分。

    中国で買った CD(シーディー) を　友達 (ともだち) に　貸 (か) しました。 我把在中国买的 CD 借给朋友了。
    私が知らない人 (ひと)　から　手紙が来 (き) ました。 一个不认识的人给我来了信。

    注：修饰名词的动词小句中的主语，不能用は必须用が表示。

    私が知らない人 (ひと)（我不认识的人）
    森さんが買った本（森先生买的书）

    # [小句 (一类性 / 二类形 / 名)] + [名]

    • “名词 + が + 一类形容词 + です” 修饰名词时，一类形容词要用基本形。
    • “名词 + が + 二类形容词 + です” 修饰名词时，“二类形容词 + です” 变成 “二类形容词加な”。
    • “名词 1 + が + 名词 2 + です” 修饰名词时，“名词 2 + です” 变成 “名词 2 + の”。

    入 (い) り口 (ぐち) が広い あの建物 (たてもの) が私の会社です。 入口很大的那座楼是我们的公司。（形 1 基本形 修饰名词）

    操作 (そうさ) が簡単な パソコンが欲しいです。 我想要操作简单的个人电脑。 （形 2 + な 修饰名词）

    数学 (すうがく) が専門の 先生 (せんせい) が休みました。 教数学的老师休息了。 （名词 + の 修饰名词）

    注：専門 (せんもん)

    形容词修饰名词不能加 “の”。

    # 表达

    1. ～のところ
      名词后加 “～のところ” 的形式，可以把本来不表示场所的名称变为表示场所的名称。 即 “～那里”。

    あの　窓 (まど) のところ　にいる人はだれですか。 在窗户那儿的人是谁啊？

    1. あれ [指人的词]
      “これ / あれ” 本来是指示事物的词，但有时也用于指人，不过礼貌程度较低，一般用于指示或介绍自己的家人或部下，当然如果所指的人不在场，或者在远处而听不到谈话时，可以使用。

    2. ～でしたね
      “～でしたね” 用于表示确认。“对吧，是吗”。
      “～でしたね” 表示确认时，可以与副词 “たしか” 呼应使用。

    日本と中国の時差 (じさ) は１時間でしたね。 日本和中国的时差时 1 小时来着，对吗？
    —— ええ，日本のほうが１時間早 (はや) いです。 对，日本早一小时。

    1. 大 (おお) きな
      和 “大きい”“小さい” 表示同样意思的还有 “大 (おお) きな”“小 (ちい) さな” 两个词，不过 “大 (おお) きな”“小 (ちい) さな” 只能用于修饰名词（不能作谓语，如 “～は　大 (おお) きなです），不能用来结句。其他的一类形容词没有这种情况。
      小さなかばんが欲しいです。 想要小的提包。

    2. このあたり
      “あたり” 表示某一场所的周边，除可接在 “この / その / あの / どの” 等词后面外，还可接在一些表示具体场所的名词后面，如 “新宿 (しんじゅく) あたり（新宿一带）”。

    このあたりはよく渋滞 (じゅうたい) します。 这一带经常堵车。

    # 第二十六课：自転車に 2 人 (ふたり) で乗るのは危 (あぶ) ないです

    # 语法

    # [小句 (动词简体形)] + の + は [形]　です

    动词小句加の使其名词化，表示 “做某动作” 的意思。这种名词化形式作主语而谓语是表示性质、状态的一类形容词或二类形容词是，主语要用助词 “は” 表示。

    パソコンで表 (ひょう) を作るのは　楽 (たの) しいです。 用电脑制表是很愉快的。

    • 注：如果形容词是 “好 (す) きです”“嫌 (きら) いです”“上手 (じょうず) です”“下手 (へた) です” 等主观性很强烈的形容词，如果名词化形式是这些形容词的对象时，要用助词 “が”。
      私は　自転車に乗るのが　下手です。 我骑车技术不行。

    • 注：“好 (す) きです” 等表示好恶的形容词的对象作为话题使用时，要用助词 “は” 表示。
      絵えをかくのは好きですか。 画画你喜欢吗？
      —— いいえ，絵をかくのは嫌いです。 不，画画我不喜欢。

    如果是评价某人对于什么什么 是什么感情，用 “が”；如果是讨论话题，用 “は”。

    # [小句 (动词简体形)] + の　+ を　[动]

    动词小句加 “の” 的名词化形式还可以做宾语。

    手紙を出 (だ) すのを　忘 (わす) れました。 我忘了寄信。
    森さんが発言 (はつげん) するのを　聞 (き) きました。 听了森先生的发言。

    注：这种名词化形式作宾语时 “の” 和 “こと” 可以互换。

    • 不过句尾的动词时 “知らせます（告诉）”“伝 (つた) えます（告诉）”“話 (はな) します（说）” 等表示语言行为的动词时，只能用 “こと”。
      李さんは森さんが休むことを伝えました。 小李转告了森先生请假的事情。

    • 句尾的动词时 “見ます（看）”“聞きます（听）” 等表示感知的动词或 “手伝 (てつだ) います（帮忙）”“やめます（停止）”“防 (ふせ) ぎます（防御）” 等表示直接作用于动作对象的动词时，只能用 “の”。
      李さんは森さんが車から降 (お) りるのを見ました。 小李看见森先生从车上下来了。

    # [小句 (简体形)] でしょう [推测]

    表示说话人对自己和听话人都不能断定的事情进行推测，常与 “たぶん” 呼应使用。动词小句和一类形容词小句用简体形后续 “でしょう”，二类形容词小句和名词小句则把简体形的 “だ” 换成 “でしょう” 即可。“でしょう” 的简体形是 “だろう”。

    明日に朝 (あさ) は大雨 (おおあめ) になるでしょう。 明天早晨会下大雨吧。 （动词）

    この本の値段 (ねだん) は分かりませんが，たぶん高いでしょう。 我不知道这本书的价格，大概很贵吧。 （一类形容词）

    桜さくらの季節きせつですから，京都はたぶんにぎやかでしょう。 现在正是观赏樱花的季节，京都大概很热闹吧。 （二类形容词）

    今年 (ことし) はたぶん豊作 (ほうさく) だろう。 今年大概是个丰收年吧。 （名词）

    # [小句 (简体形)] かもしれません

    表示可能发生某事，其可能性为 50% 左右。动词小句和一类形容词小句用简体形后续 “かもしれません”，二类形容词小句和名词小句则把简体形的 “だ” 换成 “かもしれません” 即可。 同上一点的语法

    明日は雪かもしれません。 明天也许下雪。

    注：与 “～でしょう” 相比，“～かもしれません” 表示的概率较低。

    可以在同一动作的肯定形和否定形后面并列使用两个 “かもしれません”。
    明日は雨が降るかもしれませんが，降らないかもしれません。 明天也许下雨，也许不下雨。

    # 表达

    1. もしかしたら
      口语形式，用于可能性较低的场合。一般与 “～かもしれません”“～ではありませんか” 等呼应使用。
      馬さんはまだ来ませんね。 小马还没来啊？
      —— もしかしたら，今日は来ないかもしれませんよ。 也许今天不来了。

    2. それで
      表示前面句子的事态为后面句子事态的原因、理由。
      日本には握手 (あくしゅ) の習慣 (しゅうかん) がありません。それで，つい握手 (あくしゅ) するのを忘れます。 日本没有握手的习惯，因此，一部注意就忘记握手了。

    这里以对方叙述的事情为理由，说话人使用 “それで” 引出其导致的结果。

    注：“だから（所以）” 也表示原因、理由。但与 “それで” 相比，语气较强，且后面可以是表示祈使或推测的句子，而 “それで” 不行。
    すぐに会議が始まります。だから，急 (いそ) いでください。 会议就要开始了，所以你快点吧。 （祈使要求，不能用それで）

    1. つい
      表示并非有意所为却形成了某种后果，并含有对意想不到的后果感到不快、后悔的语感。

    2. あいさつ回 (まわ) り
      指到了新单位或迁入新居后，去拜访有关单位或周围邻居的行为。本来是 “あいさつして回ります（寒暄拜访）”，但现在多用 “あいさつ回 (まわ) り” 这种名词形式，如 “あいさつ回 (まわ) りをします”“あいさつ回 (まわ) りに行きます”。

    # 第二十七课：子供の時，大きな地震 (じしん) がありました

    # 语法

    # [小句 (简体形)] + 時

    “小句 + 時” 是表示时间的名词短语。 相当于动词、名词、形容词和名词连接时的语法

    1. 小句为动词小句是分两种情况：
    • “～する（基本形）+ 時” 表示后项动作实现时前项动作尚未完成。
    • “～した（た形）+ 時” 表示后项动作实现时前项动作已完成。
    1. 小句为一类形容词小句时：形 1 简体形 + 時
    2. 小句为二类形容词或名词小句时：形 2 + な + 時，名词 + の + 時

    对于 2 和 3，前项是后项的动作进行的时间。

    子供の時，大きな地震 (じしん) がありました。 我小时候，发生过大地震。
    日本に行く　時，たくさんお土産を買いました。 去日本的时候买了许多礼物。（基本形，还没到日本买的）
    日本に行った　時，たくさんお土産を買いました。 去了日本的时候买了很大礼物。 （た形，去了日本才买的）
    忙しい　時，家族みんなで仕事をします。 忙时全家人一起干活。
    暇 (ひま) な　時，私は町 (まち) で買い物をします。 有空时我上街买东西。

    注：只有名词作修饰语时，如 “子供の時” 才在名词 “子供” 与 “時” 之间加 “の”，其他词语作修饰语则一律不加 “の”，如 “忙しいの時” 是错误的用法。

    表示一次性的、个别的情况时，也可以在 “～時” 后面加 “に”。
    この前，友達が病気 (びょうき) の時に，私が看病 (かんびょう) しました。 前些天朋友生病时我护理他了。

    # [动]　ながら

    表示同一主体进行两个动作，其中后面的动作是主要动作。“ます” 形去掉 “ます” 加 “ながら”。

    李さんはテレビを見ながら食事 (しょくじ) をしています。 小李边看电视边吃饭。

    # [小句 (简体形)] でしょう？ [确认]

    如果是确认对方比自己更为熟悉的事情时，“～でしょう” 读升调，含有说话人请听话人告诉自己某种信息的语感。
    如果是用于对方和自己意见不同，或者叮问对方时，读降调。

    注：“ね” 也可用于向对方进行确认，但 “～ね” 仅仅用于说话人认为自己和听话人双方意见一致的情况。而 “～でしょう” 可用于对方和自己意见不一致时。所以一般意见一致时用 “ね”。

    # [动] ています [反复][习惯]

    之前 15、16 课学过 “动词て形 + います” 表示动作正在进行和动作的结果、状态的用法。
    这里还可以表示反复或习惯性的动作，在表示习惯性动作时可以用动词ます形代替。

    葉子 (ようこ) さんはアルバイトをしながら学校に通 (かよ) っています。 叶子边打工边上学。 [反复]
    私は毎日散歩しています。 我每天散步。 [习惯]
    私は毎日散歩します。 我明天散步。 [习惯，但用ます形表示]

    注：表示习惯性动作时，二者都可以使用，但句子里如果有使用 “先月 (せんげつ) から” 这种表示时间起点的形式时，一般不用 “ます”。
    私は先月から毎朝太極拳 (たいきょくけん) をしています。 我从上个月开始每天早上打太极拳。

    注：表示现在正在从事的工作，有时可用 “[表示职业的名词]+ を + しています” 的形式。
    私は高校 (こうこう) の教師 (きょうし) をしています。 我在做高中教师。

    # [名] で [原因、理由]

    仕事で，楊 (よう) さんと会 (あ) っていた んですよ。 因工作我和杨先生见面来着。
    小语法：

    • 会 (あ) っていた
      动词「会う」的て形 + いる（过去式）。
      「会う」是 “见面” 的意思，て形「会って」后接「いる」表示动作的持续或结果状态。
      「会っていた」是「会っている」的过去式，表示过去某段时间内的持续状态，即 “当时正在见面” 或 “当时见过面”。
    • んですよ
      「んです」是「のです」的口语形式，用于解释或强调原因。
      「よ」是语气助词，用于加强语气，表示说话人向听话人传递信息或强调某事。

    # [名] と会います

    第 8 课学习了 “～に会います” 中的 “に” 表示对象的用法，而使用表示互动行为的动词时，进行同一动作的另一方要用 “と” 表示。如 “結婚 (けっこん) します（结婚）”“けんかします（吵架）” 等表示互动行为的动作只能用 “と”。
    “会います” 既能用に也能用と，但と时含有互动的语感。

    田中さんは佐藤さんと結婚しました。

    在 “A は B と～” 中，A 为主语，而在 “A と B とが～” 中，A 和 B 都是主语（A と B と中的第二个と有时省略不用。）
    田中さんと佐藤さん（と）が結婚しました。

    # 表达

    1. アルバイト
      “打工”，日语中多指学生利用课外时间去工作的情况。

    2. 学校に通っています
      “通 (かよ) います” 表示定期往返于同一个场所，与表示目的地的 “に” 一起使用；“行きます” 也和 “に” 一起使用，但只表示从一个地方移动到另一个地方，没有往返的意思。

    3. 大勢 (おおぜい)
      表示人多，如 “大勢 (おおぜい) の人（很多人）”“人が大勢 (おおぜい) ます（有许多人）”。用于修饰名词、动词而一般不能用来做谓语（不能 “人が大勢です”）。
      “たくさん” 也可用来表示人多，但同时可用于人以外的事物，但 “大勢 (おおぜい)” 只能用于人。

    今朝 (けさ)，公園を散歩している時，大勢 (おおぜい) の人が集 (あつ) まっているのを見ました。 我今天早晨在公园散步时，看见许多人聚在一起。

    1. お年 (とし) 寄 (よ) り
      “年 (とし) 寄 (よ) り” 前加上敬语的 “お”，是既有敬意又有亲切感的 “老人” 的说法，但是，“お年 (とし) 寄 (よ) り” 不能直接用来称呼对方。 直接称呼时用 “おじいさん（老爷爷）”“おばあさん（老奶奶）”
      お年 (とし) 寄 (よ) りが多 (おお) かったでしょう？ 老年人很多吧。

    2. そう言 (い) えば
      “说起来”，表示由正在进行的对话或说话现场的某种情况联想到另外一些情况时使用。

    3. へえ [叹词]
      表示对听起来的信息感到敬佩或惊讶。

    # 第二十八课：馬さんはわたしに地図をくれました

    # 语法

    # [名 1 (人)] は　[名 2 (人)] に　[名 3 (物)] をくれます

    “くれます” 表示别人给说话人或说话人一方的人某物。

    辨析：
    あげる：1. 我给别人 2. 别人给别人
    くれる：1. 别人给我 2. 别人给我这边的人（亲人等）

    小野さん，この本をわたしにくれますか。 小野，这本书送给我吗？ （别人给我）
    昨日，馬さんが妹 (いもうと) に旅行のお土産をくれました。 昨天，小马把旅行时买的礼物给了我妹妹。 （别人给我的妹妹）

    注：日语中，一般采用以说话人作主语进行叙述的说法。如，相比于 “馬さんがわたしに本をくれました。”，更常用 “わたしは馬さんに本をもらいました。”

    # [动] てあげます

    即 “动词て形 + あげます”，表示说话人或说话人一方的人为别人做某事。“别人” 用助词 “に” 表示。

    森さんはお年寄 (としよ) りの荷物 (にもつ) を　持 (も) って　あげました。 森先生帮老年人拿行李了。
    おじいさんが孫 (まご) に本を　読んで　あげました。 爷爷念书给孙子听。

    # [动]　てもらいます

    即 “动词て形 + もらいます”，具有 “说话人请别人做某事” 以及 “说话人承受了由于别人的动作而带来的恩惠” 两种含义。“别人” 用助词 “に” 表示。

    森さんは李さんに北京を　案内 (あんない) して　もらいました。 森先生请小李带他游览了北京。
    係 (かかり) の人に切符 (きっぷ) を　交換 (こうかん) して　もらいました。 请工作人员给我换了票。

    林 (はやし) さん，もう少 (すこ) し　待って　もらいたいのですが。 林先生，我想请你再稍微等一等。
    —— 分かりました。 知道了。

    # [动] てくれます

    即 “动词て形 + くれます”，表示说话人以外的主语为说话人或说话人一方的人做某事（别人为我或我这边的人做某事）。如果整个句子为疑问形式时也可用于委托关系亲密的人为自己做某事。

    女 (おんな) の人がわたしの財布 (さいふ) を　拾 (ひろ) って　くれました。 一位妇女帮我捡起了钱包。
    ごみを　出 (だ) して　くれますか。 —— いいですよ。 帮我倒一下垃圾好吗？ —— 好的。

    委托别人帮自己做某事时，使用否定形式 “～てくれませんか” 更礼貌。
    森君，この手紙をコピーしてくれますか。 森君，能帮我把这封信复印一下吗？

    # 表达

    1. ～てあげます
      “～てあげます” 表示说话人或者说话人一方的人为别人做某事，但含有赐恩于对方的意思，不太礼貌，不能面对面直接对尊长使用。
      对尊长说话时，用征求对方意见的形式 “～ましょうか”。

    2. それに
      “而且”。用于前后句的内容为累加关系，多用于口语。

    3. ～先 (さき)
      表示移动性动作的到达地点或归属。如 “引 (ひ) っ越 (こ) し先（搬迁的地址）” 指搬迁后居住的地方。

    出張 (しゅっちょう) 先のホテルから電話がありました。 从出差地的宾馆打来了电话。

    其他例如：行き先（目的地，要去的地方）；旅行先（旅行的目的地）；就職 (しゅうしょく) 先（就业的单位）；送 (おき) り先（邮件的接受人和地点）

    1. どの辺 (へん)
      询问大致的场所时使用。回答时回答一个大致的场所，也可以只回答地名。
      甲：引っ越し先が決 (き) まりました。 （搬迁的地址定下来了。）
      乙：どの辺ですか。 （在哪儿？）
      甲 ：新宿です。 （在新宿。）

    注：类似 “～あたり”，“～辺” 也表示周边的意思，但 “辺” 必须和 “この / その / あの / どの” 等一起使用，不能加在表示具体场所的名词后。

    1. 家具 (かぐ) はどうしたんですか
      日语的 “どうしたんですか” 可以用于不同的场面表示多种意思，在这里表示询问某物的购入方式。
      家具 (かぐ) はどうしたんですか。 家具怎么样了。
      —— もう買いました。明日届 (とど) けてもらいます。 已经买好了，明天送来。
      そのチケット，どうしたんですか。 入场券怎么搞到手的？
      —— インターネットで買ったんです。 在网上买的。
