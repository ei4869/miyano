    # 总结

    1. 对于 LED_NUM 为 1 时，进行了特殊处理，此时该 LED 会按一定间隔闪烁。

    2. clog2 函数能用于计算 INTERVAL 的对数值（以 2 为底）。$clog2 函数返回一个整数值，该值是大于或等于 INTERVAL 的对数值的最小整数。换句话说，它计算的是 INTERVAL 需要多少位二进制位来表示，从而确定计数器的位宽。

    1. .led(led) 和 .led(led [LED_NUM-1:0])
    • .led (led)：直接连接信号到端口，前提是信号的宽度与端口的宽度一致。
    • .led (led [LED_NUM-1:0])：明确指定信号的位宽，通常在信号宽度是参数化的情况下使用。
